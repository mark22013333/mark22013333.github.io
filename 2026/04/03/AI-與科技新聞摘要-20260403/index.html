Pinterest е·ҘзЁӢеңҳйҡҠжңҖиҝ‘йҮӢеҮәдәҶд»–еҖ‘зҡ„з”ҹз”ўзҙҡ MCP з”ҹж…Ӣзі»зөұзҙ°зҜҖпјҢжңүеӨ жүҜж¬ёгҖӮд»–еҖ‘дёҚжҳҜеҸӘжҗһеҖӢзҺ©е…·еҮәдҫҶпјҢиҖҢжҳҜжү“йҖ дәҶе®Ңж•ҙзҡ„еҹәзӨҺиЁӯж–ҪпјҢж•ҙеҗҲ PrestoгҖҒSparkгҖҒAirflow зӯүй ҳеҹҹзү№е®ҡзҡ„дјәжңҚеҷЁпјҢй…ҚдёҠдёӯеӨ®иЁ»еҶҠиЎЁе’Ңдәәе·ҘеҜ©ж ёжІ»зҗҶж©ҹеҲ¶гҖӮ

ж•ёеӯ—жңғиӘӘи©ұпјҡжңҲеқҮ 66,000 ж¬Ўе‘јеҸ«гҖҒ844 еҖӢжҙ»иәҚдҪҝз”ЁиҖ…пјҢжҜҸеҖӢжңҲе№«е…¬еҸёзңҒдёӢ 7,000 е°ҸжҷӮгҖӮйҖҷиЎЁзӨә MCP е·Із¶“еҫһгҖҢжңүи¶Јзҡ„еҜҰй©—гҖҚеҚҮзҙҡжҲҗзңҹеҜҰз”ҹз”ўзі»зөұпјҢй–Ӣе§ӢеңЁеӨ§иҰҸжЁЎе·ҘзЁӢеңҳйҡҠзҡ„ж—Ҙеёёе·ҘдҪңжөҒдёӯзҷјжҸ®еҜҰйҡӣеғ№еҖјгҖӮйҖҷе°Қж•ҙеҖӢ MCP з”ҹж…Ӣзҷјеұ•жҳҜеҖӢжӯЈйқўдҝЎиҷҹгҖӮ

Chrome йӣ¶ж—ҘжјҸжҙһ CVE-2026-5281 йҒӯеҜҰйҡӣеҲ©з”Ё

Google еҸҲжҢЁдәҶдёҖеҲҖгҖӮCVE-2026-5281 жҳҜ DawnпјҲWebGPU еә•еұӨеј•ж“ҺпјүиЈЎзҡ„ use-after-free жјҸжҙһпјҢзҸҫеңЁе·Із¶“иў«еҜҰйҡӣеҲ©з”ЁпјҢ4 жңҲ 1 ж—Ҙиў« CISA еҠ е…Ҙе…¬й–ӢжјҸжҙһзӣ®йҢ„гҖӮеҪұйҹҝзүҲжң¬еҢ…жӢ¬ Chrome < v146.0.7680.177/178гҖӮ

йҖҷжҳҜ Google 2026 е№ҙзҲҶеҮәзҡ„з¬¬ 4 еҖӢйҒӯеҜҰйҡӣеҲ©з”Ёзҡ„йӣ¶ж—ҘжјҸжҙһгҖӮзҖҸиҰҪеҷЁе®үе…Ёзҡ„и»ҚеӮҷз«¶иіҪзңҹзҡ„жІ’е®ҢжІ’дәҶпјҢз”ЁжҲ¶жңҖеҜҰйҡӣзҡ„йҳІе®Ҳе°ұжҳҜдҝқжҢҒжӣҙж–°гҖӮ

Invariant Labs жҸӯйңІдәҶдёҖеҖӢ MCP з”ҹж…Ӣзҡ„ж–°еЁҒи„…пјҡе·Ҙе…·жҠ•жҜ’ж”»ж“ҠпјҲTool PoisoningпјүгҖӮй§ӯе®ўеҸҜд»ҘеңЁ MCP е·Ҙе…·жҸҸиҝ°иЈЎеҹӢе…ҘжғЎж„ҸжҢҮд»ӨпјҢAI Agent жңғзӣҙжҺҘеҹ·иЎҢпјҢе°ҺиҮҙиЎҢзӮәиў«еҠ«жҢҒгҖҒиіҮж–ҷеӨ–жҙ©пјҢз”ҡиҮідҝЎд»»зҡ„дјәжңҚеҷЁйғҪиў«жҺЁзҝ»гҖӮжңҖжҒҗжҖ–зҡ„ең°ж–№жҳҜйҖҷдәӣйҡұи—ҸжЁҷзұӨе°ҚдҪҝз”ЁиҖ…дёҚеҸҜиҰӢпјҢдҪҶ AI жЁЎеһӢжңғиҷ•зҗҶе®ғеҖ‘гҖӮ

йҖҷиё©еҲ°дәҶ AI Agent дҫӣжҮүйҸҲе®үе…Ёзҡ„й—ңйҚөз—ӣй»һгҖӮйҡЁи‘— Agent и¶ҠдҫҶи¶ҠеӨҡдәәз”ЁпјҢж•ҙеҖӢе·Ҙе…·з”ҹж…Ӣзҡ„дҝЎд»»жЁЎеһӢйңҖиҰҒйҮҚж–°еҜ©иҰ–гҖӮйҖҷдёҚеҸӘжҳҜ Pinterest иҰҒиҖғж…®зҡ„е•ҸйЎҢпјҢжүҖжңүжҺЎз”Ё MCP зҡ„зө„з№”йғҪиҰҒе°ҸеҝғйҳІзҜ„гҖӮ
