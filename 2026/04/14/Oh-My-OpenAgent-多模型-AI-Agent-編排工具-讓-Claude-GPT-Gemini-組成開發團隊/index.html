Oh My OpenAgent - 多模型 AI Agent 編排工具，讓 Claude/GPT/Gemini 組成開發團隊

發表於2026-04-14|更新於2026-04-14|AI與ML
|總字數:1.1k|閱讀時間:4分鐘|瀏覽量:

50,816 顆星，建立不到五個月，99% 的程式碼是 AI 自己寫的。

Oh My OpenAgent（簡稱 OmO）做了一件其他 AI coding 工具不太敢做的事：把 Claude、GPT、Gemini、Grok 這些互為競爭對手的模型，硬是塞進同一個工作流裡協作。它的出發點很實際 - Claude 擅長遵循指令、GPT 擅長深度推理、Gemini 擅長視覺前端。沒有哪一個模型在所有任務都最強，那幹嘛只用一個？

不是框架，是已經調好的團隊

跟 LangChain 或 CrewAI 不同，OmO 不是讓你「建構 Agent」的框架。它是一個跑在 OpenCode 上的 Plugin，開箱即用就有一整支 AI 開發團隊：

Agent 角色 綁定模型
Sisyphus 專案經理，接需求、拆任務、分配 Claude Opus 4.6
Prometheus 架構師，會「面試」你來釐清需求 Claude Opus 4.6
Hephaestus 資深工程師，啃最難的邏輯問題 GPT-5.4
Explore Junior，快速翻 codebase 找東西 Grok
Momus Code reviewer，專門挑毛病 GPT-5.4
Librarian 文件搜尋 MiniMax

這些 Agent 之間的溝通不是指定模型名稱，而是用語義路由（Category）。比如 visual-engineering 自動走 Gemini 3.1 Pro，ultrabrain 走 GPT-5.4 高配模式。你不用管背後是誰在做事，OmO 會自動分配。

Agent 會自動探索你的 codebase、研究既有模式、實作功能、跑測試驗證，一路做到完才停。

Hash-Anchored Edit：解決 AI 編輯程式碼的老問題

傳統 AI 編輯程式碼用行號定位，但中間插了一行，後面全偏移，edit 就炸了。這個 stale-line 問題困擾 AI coding 工具很久。OmO 的 Hashline 機制給每一行附上 content hash，用 hash 而不是行號定位。

實測數據：Grok 模型的編輯成功率從 **6.7% 飆到 68.3%**。十倍的提升，這不是微調，是架構層面的改進。

多 Agent 平行工作

可以同時跑 5 個以上的 Agent 做不同的事。一邊改後端 API、一邊寫前端元件、一邊跑測試。搭配內建的 LSP + AST-Grep 支援（workspace rename、go-to-definition、find-references），跨 Agent 的學習會自動傳遞，改了 A 不會忘了 B。

大型功能可以用規劃模式：按 Tab 切到 Prometheus Agent，它會像面試一樣問你問題釐清範圍，產出計畫後交給 Atlas 排程、分配任務給多個 Agent 平行執行。

跟 Claude Code 比一下

面向 Oh My OpenAgent Claude Code
定位 多模型 Agent 編排 單模型 CLI Agent
多模型 核心特色，自動路由 僅 Anthropic 模型
Agent 分工 11 個預設 Agent 單一 Agent
平行執行 5+ Agent 同時工作 無（除非用 sub-agents）
使用門檻 需裝 OpenCode + Plugin 直接裝就能用

Claude Code 的優勢在簡單直接，一個指令就能跑。OmO 的優勢在複雜場景下的分工協作。兩者不是非此即彼的關係。

要注意的事

授權不是 MIT。OmO 用 SUL-1.0（Sustainable Use License），僅限內部商業用途或非商業個人用途。想拿它做產品賣，要特別看清楚授權條款。

多模型意味著多帳號。要完整發揮功能，至少需要 Claude 訂閱加 ChatGPT 訂閱。作者推薦的最低組合：ChatGPT $20/月 + Kimi Code $0.99/月 + GLM $10/月。

Sisyphus 強烈綁定 Claude Opus 4.6，換其他模型當主編排器，體驗會明顯下降。Token 消耗也偏高 - 多 Agent 平行工作就是在多燒 token，作者認為這是生產力的合理投資。

從去年 12 月建立到現在不到五個月，版本迭代平均每 2-3 天一版。正在從 oh-my-opencode 改名為 oh-my-openagent，過渡期兩個名字都能用。

這個專案代表了 AI coding 工具的一個有趣方向：與其押寶某一個模型，不如讓每個模型做它最擅長的事。至於這條路走不走得通，五萬顆星至少說明了開發者社群投了贊成票。
