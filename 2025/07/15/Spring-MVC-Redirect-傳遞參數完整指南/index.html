еңЁ Spring MVC иЈЎйқўеҒҡ redirect еҶҚеёёиҰӢдёҚйҒҺпјҢдҪҶжҖҺйәје®үе…Ёең°еӮійҒһеҸғж•ёпјҢе°ұжңүй»һи¬ӣз©¶дәҶгҖӮд№ӢеүҚжңүеҖӢе°ҲжЎҲеӣ зӮә redirect жҷӮжІ’иҷ•зҗҶеҘҪж•Ҹж„ҹиіҮиЁҠпјҢе·®й»һеҮәдәӢгҖӮд»ҠеӨ©е°ұе®Ңж•ҙд»Ӣзҙ№е№ҫзЁ®еӮійҒһеҸғж•ёзҡ„ж–№ејҸгҖӮ

е•ҸйЎҢиғҢжҷҜ

еҒҮиЁӯз”ЁжҲ¶жҸҗдәӨдәҶдёҖеҖӢиЎЁе–®пјҢдҪ зҡ„ controller иҷ•зҗҶе®ҢеҫҢжғіиҰҒ redirect еҲ°еҸҰдёҖеҖӢй ҒйқўгҖӮдҪҶжңүжҷӮеҖҷдҪ йңҖиҰҒеӮідёҖдәӣиіҮж–ҷзөҰдёӢдёҖеҖӢй ҒйқўпјҢжҜ”еҰӮжҲҗеҠҹ/еӨұж•—зҡ„иЁҠжҒҜпјҢжҲ–иҖ…дҪҝз”ЁиҖ… IDгҖӮ

жңҖз°Ўе–®зҡ„ж–№ејҸе°ұжҳҜеҠ еҲ° URL query stringпјҡ

дҪҶйҖҷжЁЈжңүе•ҸйЎҢпјҡ

  1. ж•Ҹж„ҹиіҮиЁҠжңғжҡҙйңІеңЁ URL дёҠ
  2. з”ЁжҲ¶еҸҜд»ҘзӣҙжҺҘж”№ URL query string дҪңејҠ
  3. еҰӮжһңеҸғж•ёеҫҲеӨҡпјҢURL жңғеҫҲй•·еҫҲйҶң

жӣҙеҘҪзҡ„ж–№ејҸжҳҜз”Ё RedirectAttributesгҖӮ

RedirectAttributes зҡ„е…©зЁ®ж–№ејҸ

ж–№ејҸдёҖпјҡaddAttributeпјҲеҠ еҲ° URL query stringпјү

йҖҷжЁЈжңғ redirect еҲ° /order/confirm?orderId=12345&status=successгҖӮ

з”ЁйҖ”пјҡеӮійҒһдёҚж•Ҹж„ҹзҡ„гҖҒеҸҜд»Ҙиў«дҪҝз”ЁиҖ…зңӢеҲ°зҡ„иіҮиЁҠгҖӮ

ж–№ејҸдәҢпјҡaddFlashAttributeпјҲеӯҳеңЁ sessionпјҢдёҖж¬ЎжҖ§пјү

redirect д№ӢеҫҢпјҢдёӢдёҖеҖӢ request еҸҜд»ҘзӣҙжҺҘеңЁ Model иЈЎжӢҝеҲ°пјҡ

жҲ–иҖ…еңЁ Thymeleaf template иЈЎйқўзӣҙжҺҘз”Ёпјҡ

addAttribute vs addFlashAttribute

зү№жҖ§ addAttribute addFlashAttribute
еӯҳж”ҫдҪҚзҪ® URL query string HTTP session
еҸҜиҰӢжҖ§ еңЁ URL дёҠеҸҜиҰӢ дёҚеңЁ URL дёҠ
з”ҹе‘ҪйҖұжңҹ ж°ёд№…пјҲзӣҙеҲ°жүӢеӢ•ж”№пјү дёҖж¬ЎжҖ§пјҲredirect еҫҢе°ұж¶ҲеӨұпјү
дҪҝз”Ёе ҙжҷҜ зҜ©йҒёгҖҒеҲҶй ҒзӯүеҸғж•ё иЁҠжҒҜгҖҒйҖҡзҹҘжҸҗзӨә
ж•Ҹж„ҹиіҮиЁҠ дёҚйҒ©еҗҲ йҒ©еҗҲ

еҜҰеӢҷдҫӢеӯҗ

еҒҮиЁӯдҪ жңүеҖӢеҲӘйҷӨиЁӮе–®зҡ„еҠҹиғҪпјҡ

Template иЈЎйқўпјҡ

еӣӣзЁ® Redirect зҡ„ж–№ејҸ

йҷӨдәҶ addAttribute е’Ң addFlashAttributeпјҢSpring MVC йӮ„жҸҗдҫӣдәҶе…¶д»– redirect зҡ„ж–№ејҸгҖӮ

ж–№ејҸдёҖпјҡString ж–№ејҸпјҲжңҖеёёз”Ёпјү

ж–№ејҸдәҢпјҡRedirectView

setContextRelative зҡ„дҪңз”ЁжҳҜжҠҠ redirect зҡ„и·Ҝеҫ‘з•¶жҲҗзӣёе°Қж–јжҮүз”Ёзҡ„ context pathгҖӮеҰӮжһңдҪ зҡ„жҮүз”Ёзҡ„ context path жҳҜ /myappпјҢcontextRelative(true) е°ұжңғ redirect еҲ° /myapp/usersпјӣcontextRelative(false)пјҲй җиЁӯпјүеүҮжңғ redirect еҲ° /usersгҖӮ

ж–№ејҸдёүпјҡModelAndView

ж–№ејҸеӣӣпјҡHttpServletResponse

жңүжҷӮеҖҷдҪ жғіиҰҒжӣҙзҙ°зІ’еәҰзҡ„жҺ§еҲ¶пјҢеҸҜд»ҘзӣҙжҺҘз”Ё HttpServletResponseпјҡ

з”ЁйҖҷзЁ®ж–№ејҸзҡ„жҷӮеҖҷпјҢSpring дёҚжңғеҶҚиҷ•зҗҶ viewпјҢдҪ иҰҒиҮӘе·ұиІ иІ¬ж•ҙеҖӢ responseгҖӮ

еёёиҰӢзҡ„еқ‘

1. Flash attribute иў« redirect еҗһжҺү

Flash attribute жңғиҮӘеӢ•еңЁ redirect еҫҢж¶ҲеӨұгҖӮеҰӮжһңдҪ дёҚе°ҸеҝғеҒҡдәҶеӨҡж¬Ў redirectпјҢжңғдёҹжҺүиіҮж–ҷпјҡ

и§Јжі•е°ұжҳҜеҸӘеңЁжңҖеҫҢдёҖеҖӢ step еүҚеҠ  flash attributeгҖӮ

2. addAttribute зҡ„еҸғж•ёжІ’жңүиў«жӯЈзўәз·Ёзўј

еҰӮжһңеҸғж•ёеҗ«жңүзү№ж®Ҡеӯ—е…ғпјҢеҸҜиғҪжңғиў«з ҙеЈһпјҡ

Spring жңғиҮӘеӢ•е№«дҪ з·ЁзўјпјҢжүҖд»ҘйҖҡеёёдёҚжҳҜе•ҸйЎҢгҖӮдҪҶеҰӮжһңдҪ з”Ё String.format жүӢеӢ•зө„ URLпјҢиҰҒе°Ҹеҝғпјҡ

3. POST-Redirect-GET жЁЎејҸ

еҰӮжһңиЎЁе–®жҸҗдәӨд№ӢеҫҢзӣҙжҺҘ forwardпјҲдёҚ redirectпјүпјҢдҪҝз”ЁиҖ…жҢү F5 йҮҚж–°ж•ҙзҗҶжңғйҮҚиӨҮжҸҗдәӨиЎЁе–®гҖӮжӯЈзўәзҡ„еҒҡжі•жҳҜ POST д№ӢеҫҢ redirectпјҢз„¶еҫҢ GET ж–°й Ғйқўпјҡ

URI Template и®Ҡж•ё

жңүжҷӮеҖҷдҪ жғіиҰҒ redirect еҲ°жңү path variable зҡ„ URLпјҢеҸҜд»ҘйҖҷжЁЈеҒҡпјҡ

йҮҚй»һж•ҙзҗҶ

  • addAttribute - еҠ еҲ° URL query stringпјҢйҒ©еҗҲдёҚж•Ҹж„ҹзҡ„еҸғж•ё
  • addFlashAttribute - еӯҳеңЁ sessionпјҢдёҖж¬ЎжҖ§пјҢйҒ©еҗҲиЁҠжҒҜжҸҗзӨә
  • Flash attribute еҸӘжҙ»дёҖж¬Ў redirectпјҢеҲ·ж–°й Ғйқўе°ұж¶ҲеӨұдәҶ
  • еӣӣзЁ® redirect ж–№ејҸпјҡStringгҖҒRedirectViewгҖҒModelAndViewгҖҒHttpServletResponse
  • POST д№ӢеҫҢдёҖе®ҡиҰҒ redirectпјҢдёҚиҰҒ forwardпјҢйҳІжӯўиЎЁе–®йҮҚиӨҮжҸҗдәӨ
  • еҸғж•ёз·ЁзўјдәӨзөҰ Spring иҷ•зҗҶпјҢдёҚиҰҒжүӢеӢ•зө„ URL

з”Ё RedirectAttributes жҗһе®ҡ redirect зҡ„еҸғж•ёеӮійҒһпјҢд»Јзўјжңғжӣҙд№ҫж·Ёжӣҙе®үе…ЁгҖӮ
