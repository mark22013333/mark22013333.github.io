Optional жҳҜ Java 8 еј•е…Ҙзҡ„жҰӮеҝөпјҢеҲқзңӢеҫҲз°Ўе–®пјҢе…¶еҜҰз”Ёжі•и¶…ж·ұгҖӮдёҖй–Ӣе§ӢеҫҲеӨҡдәәиҰәеҫ— Optional еҫҲз…©пјҢеҫҢдҫҶз”Ёе°ҚдәҶжүҚзҷјзҸҫгҖҢзңҹйҰҷгҖҚгҖӮжңҖй—ңйҚөжҳҜзҗҶи§Је®ғзҡ„дёүж®өејҸзөҗж§ӢгҖӮ

Optional зҡ„дёүж®өејҸзөҗж§Ӣ

Optional зҡ„з”Ёжі•еҸҜд»ҘеҲҶжҲҗдёүж®өпјҡй ӯйғЁпјҲе»әз«ӢпјүвҶ’ иә«й«”пјҲиҪүжҸӣпјүвҶ’ е°ҫе·ҙпјҲеҸ–еҖјпјү

й ӯйғЁпјҡе»әз«Ӣ Optional

of() - зўәе®ҡйқһ null

ofNullable() - еҸҜиғҪ null

empty() - жҳҺзўәзҡ„з©ә

flatMap еҺҹзҗҶпјҡ

йҮҚиҰҒпјҡorElse зҡ„й җиЁӯеҖјжңғиў«зӣҙжҺҘжұӮеҖј

orElse vs orElseGet зҡ„е·®з•°иЎЁпјҡ
