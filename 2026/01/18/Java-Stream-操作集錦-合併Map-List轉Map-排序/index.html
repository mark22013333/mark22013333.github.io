效能提示

  • 合併 Map - 如果 Map 超級大，考慮直接 putAll 而不是 stream
  • List 轉 Map - 如果 List 有上百萬元素，stream 會有開銷，直接 for loop 可能更快
  • 排序 - Stream sorted() 是 O(n log n)，大資料集要小心

但大多數時候，stream 的可讀性遠勝效能損失。除非是效能熱點，否則就用 stream 吧。
