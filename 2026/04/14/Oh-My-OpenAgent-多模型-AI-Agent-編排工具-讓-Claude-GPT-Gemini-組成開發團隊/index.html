Oh My OpenAgentпјҲз°ЎзЁұ OmOпјүеҒҡдәҶдёҖд»¶е…¶д»– AI coding е·Ҙе…·дёҚеӨӘж•ўеҒҡзҡ„дәӢпјҡжҠҠ ClaudeгҖҒGPTгҖҒGeminiгҖҒGrok йҖҷдәӣдә’зӮәз«¶зҲӯе°ҚжүӢзҡ„жЁЎеһӢпјҢзЎ¬жҳҜеЎһйҖІеҗҢдёҖеҖӢе·ҘдҪңжөҒиЈЎеҚ”дҪңгҖӮе®ғзҡ„еҮәзҷјй»һеҫҲеҜҰйҡӣ - Claude ж“…й•·йҒөеҫӘжҢҮд»ӨгҖҒGPT ж“…й•·ж·ұеәҰжҺЁзҗҶгҖҒGemini ж“…й•·иҰ–иҰәеүҚз«ҜгҖӮжІ’жңүе“ӘдёҖеҖӢжЁЎеһӢеңЁжүҖжңүд»»еӢҷйғҪжңҖеј·пјҢйӮЈе№№еҳӣеҸӘз”ЁдёҖеҖӢпјҹ

дёҚжҳҜжЎҶжһ¶пјҢжҳҜе·Із¶“иӘҝеҘҪзҡ„еңҳйҡҠ

и·ҹ LangChain жҲ– CrewAI дёҚеҗҢпјҢOmO дёҚжҳҜи®“дҪ гҖҢе»әж§Ӣ AgentгҖҚзҡ„жЎҶжһ¶гҖӮе®ғжҳҜдёҖеҖӢи·‘еңЁ OpenCode дёҠзҡ„ PluginпјҢй–Ӣз®ұеҚіз”Ёе°ұжңүдёҖж•ҙж”Ҝ AI й–ӢзҷјеңҳйҡҠпјҡ

Agent и§’иүІ з¶Ғе®ҡжЁЎеһӢ
Sisyphus е°ҲжЎҲз¶“зҗҶпјҢжҺҘйңҖжұӮгҖҒжӢҶд»»еӢҷгҖҒеҲҶй…Қ Claude Opus 4.6
Prometheus жһ¶ж§Ӣеё«пјҢжңғгҖҢйқўи©ҰгҖҚдҪ дҫҶйҮҗжё…йңҖжұӮ Claude Opus 4.6
Hephaestus иіҮж·ұе·ҘзЁӢеё«пјҢе•ғжңҖйӣЈзҡ„йӮҸијҜе•ҸйЎҢ GPT-5.4

Momus Code reviewerпјҢе°Ҳй–ҖжҢ‘жҜӣз—… GPT-5.4
Librarian ж–Үд»¶жҗңе°Ӣ MiniMax

йҖҷдәӣ Agent д№Ӣй–“зҡ„жәқйҖҡдёҚжҳҜжҢҮе®ҡжЁЎеһӢеҗҚзЁұпјҢиҖҢжҳҜз”ЁиӘһзҫ©и·Ҝз”ұпјҲCategoryпјүгҖӮжҜ”еҰӮ visual-engineering иҮӘеӢ•иө° Gemini 3.1 ProпјҢultrabrain иө° GPT-5.4 й«ҳй…ҚжЁЎејҸгҖӮдҪ дёҚз”Ёз®ЎиғҢеҫҢжҳҜиӘ°еңЁеҒҡдәӢпјҢOmO жңғиҮӘеӢ•еҲҶй…ҚгҖӮ

иЈқиө·дҫҶи·‘и·‘зңӢ

еүҚжҸҗжҳҜиҰҒе…ҲиЈқ OpenCodeпјҲзүҲжң¬ >= 1.0.150пјүпјҢз„¶еҫҢпјҡ

иЈқе®ҢеңЁ OpenCode иЈЎйқўжү“ ultraworkпјҲжҲ–зё®еҜ« ulwпјүе°ұиғҪе•ҹеӢ•ж•ҙеҖӢ Agent еңҳйҡҠпјҡ

Agent жңғиҮӘеӢ•жҺўзҙўдҪ зҡ„ codebaseгҖҒз ”з©¶ж—ўжңүжЁЎејҸгҖҒеҜҰдҪңеҠҹиғҪгҖҒи·‘жё¬и©Ұй©—иӯүпјҢдёҖи·ҜеҒҡеҲ°е®ҢжүҚеҒңгҖӮ

Hash-Anchored Editпјҡи§Јжұә AI з·ЁијҜзЁӢејҸзўјзҡ„иҖҒе•ҸйЎҢ

еӮізөұ AI з·ЁијҜзЁӢејҸзўјз”ЁиЎҢиҷҹе®ҡдҪҚпјҢдҪҶдёӯй–“жҸ’дәҶдёҖиЎҢпјҢеҫҢйқўе…ЁеҒҸз§»пјҢedit е°ұзӮёдәҶгҖӮйҖҷеҖӢ stale-line е•ҸйЎҢеӣ°ж“ҫ AI coding е·Ҙе…·еҫҲд№…гҖӮOmO зҡ„ Hashline ж©ҹеҲ¶зөҰжҜҸдёҖиЎҢйҷ„дёҠ content hashпјҢз”Ё hash иҖҢдёҚжҳҜиЎҢиҷҹе®ҡдҪҚгҖӮ

еҸҜд»ҘеҗҢжҷӮи·‘ 5 еҖӢд»ҘдёҠзҡ„ Agent еҒҡдёҚеҗҢзҡ„дәӢгҖӮдёҖйӮҠж”№еҫҢз«Ҝ APIгҖҒдёҖйӮҠеҜ«еүҚз«Ҝе…ғд»¶гҖҒдёҖйӮҠи·‘жё¬и©ҰгҖӮжҗӯй…Қе…§е»әзҡ„ LSP + AST-Grep ж”ҜжҸҙпјҲworkspace renameгҖҒgo-to-definitionгҖҒfind-referencesпјүпјҢи·Ё Agent зҡ„еӯёзҝ’жңғиҮӘеӢ•еӮійҒһпјҢж”№дәҶ A дёҚжңғеҝҳдәҶ BгҖӮ

еӨ§еһӢеҠҹиғҪеҸҜд»Ҙз”ЁиҰҸеҠғжЁЎејҸпјҡжҢү Tab еҲҮеҲ° Prometheus AgentпјҢе®ғжңғеғҸйқўи©ҰдёҖжЁЈе•ҸдҪ е•ҸйЎҢйҮҗжё…зҜ„еңҚпјҢз”ўеҮәиЁҲз•«еҫҢдәӨзөҰ Atlas жҺ’зЁӢгҖҒеҲҶй…Қд»»еӢҷзөҰеӨҡеҖӢ Agent е№іиЎҢеҹ·иЎҢгҖӮ
