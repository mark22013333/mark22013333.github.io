AI 與科技新聞摘要 20260609

發表於2026-06-09|更新於2026-06-09|AI與科技新聞
|總字數:1.7k|閱讀時間:5分鐘|瀏覽量:

當一個東西同時被搶著請進門、又被防著請出門，通常代表它變得太重要，重要到所有人都開始計算「萬一哪天少了它會怎樣」。今天這幾則新聞，從蘋果的舞台到五角大廈的機密系統，表面上各講各的，底下其實在問同一個問題：當整個產業把脖子伸進同一組模型，這條脖子有多脆。

一、蘋果 WWDC：Siri 換上 Gemini，Claude 第一次成為 iPhone 的選項

台灣時間週一深夜，Tim Cook 以執行長身分最後一次站上 Apple Park 的舞台，丟出的東西比往年都重：Siri 底層換成 Google 的 Gemini，同時推出一套「Multi-AI Extensions」機制，讓 Claude 第一次成為 iPhone 上可選的 AI；iOS 27 Beta 1 當天下午就放出來。Anthropic 那邊也補上一個 Swift 套件，讓 Apple 開發者能用帶型別的 Swift 輸出直接交棒給 Claude 做多步推理、程式生成、網頁搜尋。

蘋果一向是「軟硬整套自己掌控」的代表，現在卻把最貼身的語音入口外包給對手的模型，這個轉向本身就值得停一下。但更該注意的是那套 Multi-AI Extensions——它表面上是給使用者選擇，骨子裡是蘋果在替自己買保險。把單一模型供應商換成一個可以隨時抽換的插槽，意味著哪天 Gemini 漲價、出包、或關係生變，蘋果不必跟著一起沉。不把命押在任何一家身上，這不是優柔寡斷，是把選擇權留在自己手裡的成本控管。

原文來源：AI News Today - June 8, 2026 (buildfastwithai)

二、五角大廈想把 Claude 換掉，微軟卻把它塞進 11,000 個模型的目錄

同一週的另一端，五角大廈證實正在測試 OpenAI 和 Google 的模型，準備在機密系統裡「可能替換掉」Anthropic 的 Claude。而在商業這側，微軟剛把 Azure Foundry 的目錄擴到 11,000 個模型，Claude Opus 4.8 就掛在裡面。

把這兩件事擺在一起看會有點分裂：一邊在拔，一邊在裝。但這恰恰是看清依賴風險的好角度。換個方向問——什麼情況下「採用單一頂尖模型」會反過來變成壞決策？答案就是當你的系統重要到不能停、而供應商卻是你不能完全控制的外部單位時。機密系統要的不是「現在最強的模型」，而是「就算某一家出狀況也換得掉的架構」。國防單位這麼早就在演練替換，不是因為 Claude 不好，而是因為「換不掉」這件事本身，對他們就是無法接受的風險。商業客戶現在覺得多選項很爽，但同樣的帳，遲早每個重度使用者都得算一次。

原文來源：AI News Today - June 8, 2026 (buildfastwithai)

三、OpenAI 一手攻基因體、一手補資安：GPT-Rosalind 與 Lockdown Mode

OpenAI 這週兩個動作方向相反卻互補。六月四號的 GPT-Rosalind 更新，把基因體分析任務用掉的 token 砍了 31%，準確度反而在自家的 GeneBench 上提升——這是往「讓模型衝進專業科學領域」推。另一手則是給 ChatGPT 加了「Lockdown Mode」：開啟後會把連網能力一律關掉，包括即時瀏覽、deep research、agent 模式、檔案下載。

省 31% 的 token 是漂亮數字，但真正的訊號藏在 Lockdown Mode 這一手。一個工具會特地做「把所有對外通道一次關掉」的開關，代表開發者已經承認：模型連網、能自己行動、能下載東西——這些讓它變強的能力，同時就是它的攻擊面。能力越大，能被利用的破口也越大，這是同一枚硬幣的兩面。願意主動做一個「自廢武功」的按鈕，比起一味宣傳「我們的 agent 多全能」，反而是更成熟的姿態。

原文來源：AI News Today - June 9, 2026 roundup

四、五個前沿模型同時開賣，API 定價戰正式進入混戰

隨著 Gemini 3.5 Pro 和 Claude Sonnet 4.8 都預計月底前發布，市場上同時有五個可信的前沿模型在商業供應。結果是 API 定價的選項——以及複雜度——來到史上最高點。

對開發者來說這是好消息，但好消息底下有個容易踩的坑。當選項從「一兩家獨大」變成「五家混戰」，大家很容易陷入「每個月都在追最便宜的那家、把程式碼黏死在某家的特殊 API 上」。這就是把可替換性換成短期省的那幾塊錢。真正划算的做法，是反過來——把模型當成可以隨時抽換的零件來架，留好退路，而不是為了當月最低價，把自己鎖進一家的生態系。便宜是會變的，但「換不掉」的代價，是會在最不該出事的時候一次跟你收齊。

原文來源：AI Updates Today - June 2026 (llm-stats)

五、川普主張政府入股 AI 巨頭，Anthropic 同週罕見警告自家模型可能失控

政策面這週出現一個奇特的政治交集：川普公開支持政府入股 OpenAI、Anthropic 和 xAI，立場竟與桑德斯靠攏，而 Altman 私下也在談。幾乎同一時間，Anthropic 罕見地發出公開警告，說自家模型可能很快強到「難以控制」。

把這兩件事並排，是這週最該細想的一組。政府想入股一個產業，通常發生在它判斷這個產業「大到不能倒、或危險到不能放任」的時候——這跟當年對待大型金融機構的邏輯如出一轍。而一家公司主動跳出來說「我做的東西可能我自己也控制不住」，這在商業世界極不尋常，正常公司只會報喜。願意公開示警，要嘛是真的看到了下行風險，要嘛是想在監管落地前先卡好位置——無論哪種，都指向同一個事實：這個產業的風險，已經大到連局內人都不敢只報喜了。

下行被估值的狂歡蓋住的時候，最先說「這可能會出事」的，往往不是局外的批評者，而是手上握有最多籌碼、輸最多的那群人。他們示警，不是因為悲觀，是因為真的有東西要賠。這條線值得你接著往下盯——當局內人開始買保險，旁邊看戲的人就別再裸奔了。
