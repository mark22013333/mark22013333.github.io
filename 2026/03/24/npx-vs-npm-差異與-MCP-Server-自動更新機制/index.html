жңҖиҝ‘еңЁе№« Claude Code иЁӯе®ҡ MCP Server зҡ„жҷӮеҖҷпјҢз”ЁеҲ°дәҶ Google е®ҳж–№еҮәзҡ„ chrome-devtools-mcp йҖҷеҖӢе·Ҙе…·пјҢдё»иҰҒжҳҜжӢҝдҫҶеҒҡзҖҸиҰҪеҷЁзҡ„иҮӘеӢ•еҢ–й©—иӯүгҖӮе®үиЈқжҢҮд»ӨиЈЎз”Ёзҡ„жҳҜ npx иҖҢдёҚжҳҜ npm installпјҢдёҖй–Ӣе§ӢжҲ‘йӮ„жІ’зү№еҲҘеңЁж„ҸпјҢдҪҶеҫҢдҫҶд»”зҙ°з ”з©¶жүҚзҷјзҸҫпјҢйҖҷе…©иҖ…зҡ„иЎҢзӮәе·®з•°е…¶еҜҰзӣҙжҺҘеҪұйҹҝеҲ°зүҲжң¬жӣҙж–°зӯ–з•ҘгҖӮйҖҷйӮҠжҠҠжҲ‘ж•ҙзҗҶзҡ„зӯҶиЁҳиЁҳйҢ„дёҖдёӢгҖӮ

npm install vs npx еҲ°еә•е·®еңЁе“Әпјҹ

иӘӘеҜҰи©ұпјҢеүӣй–Ӣе§Ӣеӯё Node.js зҡ„жҷӮеҖҷпјҢжҲ‘д№ҹжҗһдёҚеӨӘжё…жҘҡ npx и·ҹ npm зҡ„е·®еҲҘпјҢзёҪиҰәеҫ—йғҪжҳҜиЈқеҘ—д»¶з”Ёзҡ„гҖӮдҪҶе…¶еҜҰе®ғеҖ‘зҡ„иЁӯиЁҲзҗҶеҝөе®Ңе…ЁдёҚеҗҢгҖӮ

з°Ўе–®дҫҶиӘӘпјҢnpm install жңғжҠҠеҘ—д»¶дёӢијүеҲ° node_modules/ иіҮж–ҷеӨҫиЈЎпјҢжҢҒд№…дҝқеӯҳеңЁжң¬ең°гҖӮиҖҢ npx зҡ„жҰӮеҝөеүҮжҳҜгҖҢиҮЁжҷӮдёӢијүгҖҒеҹ·иЎҢе®Ңе°ұеҸҜд»ҘдёҚз®ЎдәҶгҖҚгҖӮ

е№ҫеҖӢй—ңйҚөе·®з•°ж•ҙзҗҶдёҖдёӢпјҡ

  • зүҲжң¬и§ЈжһҗжҷӮж©ҹпјҡnpm install е®үиЈқжҷӮи§ЈжһҗдёҖж¬ЎзүҲжң¬е°ұеӣәе®ҡдәҶпјӣnpx еҠ дёҠ @latest зҡ„и©ұпјҢжҜҸж¬Ўеҹ·иЎҢйғҪжңғйҮҚж–°и§ЈжһҗпјҢиҮӘеӢ•жҠ“жңҖж–°зүҲ
  • зЈҒзўҹз©әй–“пјҡnpm иЈқдәҶе°ұдҪ”з©әй–“пјӣnpx з”Ёе®ҢеҸҜиғҪжңғиў«еҝ«еҸ–зӯ–з•Ҙжё…жҺү
  • е•ҹеӢ•йҖҹеәҰпјҡnpm еӣ зӮәе·Із¶“еңЁжң¬ең°дәҶжүҖд»Ҙеҝ«пјӣnpx йҰ–ж¬ЎжңғзЁҚж…ўдёҖй»һпјҲиҰҒдёӢијүпјүпјҢдёҚйҒҺеҫҢзәҢжңүеҝ«еҸ–е°ұйӮ„еҘҪ

MCP Server зҡ„е•ҹеӢ•иҲҮжӣҙж–°ж©ҹеҲ¶

е®үиЈқжҢҮд»Өй•·йҖҷжЁЈпјҡ

йҮҚй»һжҳҜйӮЈеҖӢ npx chrome-devtools-mcp@latest зҡ„йғЁеҲҶгҖӮйӮЈеҲ°еә• MCP Server д»ҖйәјжҷӮеҖҷжңғе•ҹеӢ•гҖҒд»ҖйәјжҷӮеҖҷжңғеҺ»дёӢијүжӣҙж–°е‘ўпјҹ

е•ҹеӢ•жөҒзЁӢ

ж•ҙеҖӢжөҒзЁӢеӨ§жҰӮжҳҜйҖҷжЁЈпјҡ

д»ҖйәјжҷӮеҖҷжңғи§ёзҷјдёӢијүпјҹ

йҖҷйӮҠжҲ‘зү№еҲҘж•ҙзҗҶдәҶдёҖдёӢпјҢеӣ зӮәдёҖй–Ӣе§ӢиҮӘе·ұд№ҹжҗһж··йҒҺпјҡ

  • еҗҢдёҖеҖӢ Claude Code session е…§еӨҡж¬ЎдҪҝз”Ё MCP е·Ҙе…·пјҡдёҚжңғдёӢијүпјҢMCP Server е•ҹеӢ•еҫҢе°ұеёёй§җдәҶ
  • йҮҚе•ҹ Claude Codeпјҡnpx жңғжӘўжҹҘеҝ«еҸ–пјҢйҖҡеёёдёҚжңғдёӢијүпјҲеҝ«еҸ–йӮ„еңЁзҡ„и©ұпјү
  • еҝ«еҸ–иў«жё…зҗҶжҲ–жңүж–°зүҲжң¬йҮӢеҮәпјҡйҖҷжҷӮеҖҷжүҚжңғдёӢијүдёҖж¬Ў

е№ҫеҖӢйҮҚиҰҒзөҗи«–

жҗһдәҶдёҖйҷЈеӯҗд№ӢеҫҢпјҢжҲ‘жӯёзҙҚеҮәе№ҫеҖӢйҮҚй»һпјҡ

  1. MCP Server жҳҜ Claude Code е•ҹеӢ•жҷӮе°ұи·‘иө·дҫҶзҡ„иғҢжҷҜзЁӢеәҸпјҢдёҚжҳҜжҜҸж¬Ўе‘јеҸ«е·Ҙе…·жүҚе•ҹеӢ•
  2. е№іеёёдҪҝз”Ё MCP е·Ҙе…·жҷӮпјҢеҸӘжҳҜе‘јеҸ«е·Із¶“еңЁи·‘зҡ„ MCP ServerпјҢдёҚж¶үеҸҠдёӢијү
  3. @latest зҡ„еҘҪиҷ•жҳҜиҮӘеӢ•жӣҙж–°пјҢдёҚйңҖиҰҒеҸҰеӨ–жүӢеӢ•иҷ•зҗҶзүҲжң¬жӣҙж–°
  4. жҜҸеҖӢ MCP Server д№Ӣй–“жҳҜеҗ„иҮӘзҚЁз«Ӣзҡ„пјҢжӣҙж–°дә’дёҚеҪұйҹҝ

зӮәд»ҖйәјйҒё npx иҖҢдёҚжҳҜ npm installпјҹ

е…¶еҜҰжғіжё…жҘҡд№ӢеҫҢеҺҹеӣ и »жҳҺзўәзҡ„пјҡ

  1. иҮӘеӢ•жӣҙж–° вҖ” еҠ дёҠ @latest е°ұиғҪеңЁжҜҸж¬Ўе•ҹеӢ•жҷӮи§ЈжһҗжңҖж–°зүҲпјҢе®Ңе…ЁдёҚз”ЁжүӢеӢ•з¶ӯиӯ·
  2. дёҚжұЎжҹ“е…Ёеҹҹз’°еўғ вҖ” дёҚжңғеңЁдҪҝз”ЁиҖ…зҡ„ node_modules иЈЎйқўз•ҷдёӢдёҖе ҶжӘ”жЎҲ
  3. MCP зӨҫзҫӨж…ЈдҫӢ вҖ” Claude Code е®ҳж–№ж–Үд»¶и·ҹзӨҫзҫӨзҡ„ MCP Server е№ҫд№ҺйғҪжҳҜз”Ё npx е®үиЈқпјҢз…§и‘—ж…ЈдҫӢиө°жҜ”ијғдёҚжңғиё©еқ‘

еёёз”Ёзӣёй—ңжҢҮд»Ө

жңҖеҫҢйҷ„дёҠе№ҫеҖӢжҲ‘еёёз”Ёзҡ„жҢҮд»ӨпјҢж–№дҫҝд№ӢеҫҢжҹҘпјҡ

е°ҸжҸҗйҶ’пјҡеҰӮжһңзҷјзҸҫ MCP Server иЎҢзӮәжҖӘжҖӘзҡ„пјҢеҸҜд»Ҙе…Ҳи©Ұи‘—йҮҚе•ҹ Claude CodeпјҢи®“ npx йҮҚж–°жҠ“дёҖж¬ЎжңҖж–°зүҲпјҢйҖҡеёёе°ұиғҪи§ЈжұәдәҶгҖӮ
