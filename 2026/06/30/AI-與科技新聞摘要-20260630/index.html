AI 與科技新聞摘要 2026/06/30 — 開放與封閉，這禮拜偷偷換了位置

發表於2026-06-30|更新於2026-06-30|AI與科技新聞
|總字數:2.3k|閱讀時間:7分鐘|瀏覽量:

「開源」這兩個字，這禮拜到底站在哪一邊？

如果你三年前問這個問題，答案很清楚：開放的是 Meta、是新創、是想挑戰大公司的那群人；封閉的是握著最強模型不放的巨頭。這禮拜的新聞，幾乎是把這張地圖整個翻了過來——當年扛開源大旗的轉身關門，被當成追趕者的反而接過旗子，而你天天在用的工具，有的連名字都換掉了。一條一條看，會發現它們其實在問你同一件事。

一、開源旗手 Meta，悄悄把門關上了

最反直覺的一條，來自最不該發生的地方。Meta 推出了 Muse Spark，這是 Meta Superintelligence Labs 的第一個對外發布——而它是閉源的。

這件事的份量，得放回脈絡才看得出來。過去兩年，「開源 LLM」這個詞幾乎跟 Llama 是同義詞，Meta 靠著開放權重，把自己塑造成對抗 OpenAI、Anthropic 的那一方，整個開源社群都建在 Llama 的地基上。現在第一個新品牌的旗艦，直接選了關門。產業媒體把這解讀成一次明確的轉向——不是這一代剛好閉源，是策略層級的調頭。

為什麼值得記？因為它戳破了一個很多人沒意識到的假設：你以為某家公司「開源」是它的本性，其實那只是它在某個階段、為了某個目的做的選擇。本性是會變的，尤其當賭注大到一定程度。建在別人善意上的東西，從來都不是地基，是租約。

二、旗子沒倒，只是換人扛了

旗手退場，旗子卻沒落地——接手的是一批兩年前還被當成追趕者的名字。

這禮拜開源權重這邊最重的一發是 MiniMax M3，它號稱是第一個同時把「前沿級程式能力 + 100 萬 token 上下文 + 原生多模態」三件事湊齊的開源權重模型，在開源權重的 SWE-Bench Pro 上以 59.0% 拿下第一。它不是孤例：Z.ai 的 GLM-5.2 在程式與代理能力上大幅超越前代，發布沒幾天就被 Nous Research 的 Hermes Agent 整合進去；Kimi K2.7 Code HighSpeed 主打多模態推論快 6 倍；連 Nvidia 都丟出 Nemotron 3 Ultra，被評為能力對效率比極為漂亮的一款。再加上 DeepSeek V4 Pro（MIT 授權、SWE-bench Verified 80.6%，跟閉源前沿只差 0.2 分）和全面轉 Apache 2.0 的 Mistral，前沿開放這條線，現在主要是中國實驗室加上 Mistral 在撐。

把這條跟上一條擺在一起看就有意思了：開放性沒有消失，它只是從一群人手上換到另一群人手上。這正是芒格會提醒的那種陷阱——別把「這個陣營一直會做這件事」當成永恆定律。誰開放、誰封閉，是一個會隨利益不斷重新洗牌的動態局，不是陣營的標籤。

三、阿里的兩手策略：旗艦關門，工人開放

Meta 不是唯一一個把門帶上的。阿里巴巴把 Qwen 3.7 Max 轉成了閉源——只開 DashScope 上的 API，不再放出權重。

但這條的細節才是重點：阿里沒有全關，它玩的是兩手策略。旗艦級的 Qwen 3.7 Max 收進 API 後面、變成要付費呼叫的服務；但同時，主打效率的 Qwen3-Coder 系列繼續開源、繼續放權重。一邊用最強的模型去賺錢，一邊用夠用的開源模型去佔住開發者的心智和生態。

這套操作很值得學起來看懂，因為它會變成接下來的常態：「開源 vs 閉源」很快就不會是一家公司的單選題，而是同一家公司在不同產品線上的精算。對你的意義是，下次看到某家「擁抱開源」的宣告，要先問清楚——它開放的是會下金蛋的那隻鵝，還是養來給你看的那隻？通常是後者。

四、你昨天還在用的 Windsurf，今天叫 Devin 了

模型層在洗牌，工具層也沒閒著——而且這次連名字都不留。

Windsurf 被 Cognition 收購之後，這禮拜整個併入了 Devin Desktop。windsurf.com 現在直接轉址到 devin.ai，原本免費的 Windsurf 編輯器變成 Devin 的免費方案，底層模型也從 Windsurf 自家的換成了 SWE 1.6。如果你是 Windsurf 的老用戶，某天打開來，會發現你熟悉的那個工具從名字、模型到方案結構全變了——你沒換工具，工具自己換掉了。

塔勒布式的提醒在這裡特別刺：你對一個工具的依賴，從來不只是依賴它的功能，你還默默依賴了「它明天還是它」這件事。但這件事完全不在你的掌控裡。一次收購，你習慣的操作邏輯、計價方式、甚至底層那顆模型，可能一夕之間被換成別人的決定。免費當然好，但免費工具最貴的成本，往往是它哪天不由你決定地變成另一個東西時，你被迫重新適應的那趟工。

五、計價表也在動：吃到飽的時代正在退場

換掉的不只名字，還有帳單的算法。

GitHub Copilot 從 6 月 1 日起改成用量制的 AI credits（1 credit = 0.01 美元），把原本的 premium request 制度退場，過程中還一度暫停新付費用戶註冊。Cursor 這邊則重整了 Teams 的結構，把用量拆成兩個池、加開 Premium 座位層，官方說 90% 的團隊成本會下降，新客即刻生效、續約客從 7 月 1 日起套用。

兩家動作方向不同，但底層訊號一致：那個「一個月固定多少錢、隨便你用」的吃到飽時代正在退場，換成「你用多少、算多少」。這背後是成本誠實地浮上檯面——當每一次模型呼叫的真實成本藏不住了，平台就只能把它如實轉嫁給你。對重度使用者來說，這是該認真算一次帳的時間點：你以前那種開著就不管的用法，在新制下可能會給你一張很有感的帳單。

六、最多星的 coding agent，已經不是大公司那幾個

最後一條，留給一個容易被忽略的數字。

在開源 coding agent 的星數排行上，opencode 以超過 18 萬顆星排第一，把 Claude Code（約 13.5 萬）、Gemini CLI（約 10.5 萬）、OpenAI Codex（約 9.4 萬）全壓在後面。同一段時間，AI agent 框架 Goose 移交給了 Linux Foundation 託管，OpenCode 也轉到了新的維護組織底下。

社群的星星不直接等於市佔，更不等於營收——這點得講清楚，免得過度解讀。但它確實在量一件大公司買不到的東西：開發者願不願意把心力押在一個「沒有任何一家公司能單方面收回、改價、或改名」的東西上。移交給基金會、轉成中立治理，做的就是同一件事——把「會不會哪天被某家公司收走」這個風險，從根上拆掉。

原文來源：Best AI Coding Agents (June 2026): Scored Leaderboard — Morphllm

所以這禮拜到底在教你什麼

六條看下來，「開放」和「封閉」這兩頂帽子，在不同的頭上換來換去。但與其追問「現在該選哪一邊」，不如換個更耐用的問法——下次看到任何標榜「開源」「免費」「開放」的東西，先用三個判準過一遍。

第一，這個開放，扛不扛得住一道命令或一次轉向？Fable 5 因為一紙出口管制就停了、Meta 說轉就轉、阿里旗艦說收就收——能被外力一句話關掉的開放，本質上是借來的。第二，你依賴的工具，半年後名字、計價、底層模型還是不是同一個？Windsurf 變 Devin、Copilot 改 credits 都在提醒你，今天的使用習慣，建在一個隨時會被重畫的地基上。第三，也是最關鍵的——你押注的那道「開源護城河」，到底長在你自己手上，還是租來的？

判準給完，選擇是你的。但有一件事可以先認清：開放從來不是某個工具天生的屬性，它是某個人在某個當下做的選擇，而選擇隨時可以反悔。真正穩的不是賭對哪一邊會繼續開放，是預設你依賴的每一道開放都可能哪天關上，然後反過來問自己——關上的那天，我手上還剩什麼。
