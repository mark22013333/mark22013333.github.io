Spring Bean Scope тЁГуе«СйюућетЪЪт«їТЋ┤УДБТъљ

уЎ╝УАеТќ╝2025-05-21|ТЏ┤Тќ░Тќ╝2026-05-14|Spring
|уИйтГЌТЋИ:1.2k|жќ▒У«ђТЎѓжќЊ:4тѕєжљў|уђЈУдйжЄЈ:

С╣ІтЅЇтюет░ѕТАѕУБАтЏауѓ║ТљъСИЇТИЁТЦџ Bean scope№╝їт░јУЄ┤СИђтђІтЮЉтЪІС║єтЦйС╣ЁТЅЇуЎ╝уЈЙсђѓС╣ЙУёєт░▒Тіі Spring уџётЁГуе« scope ТЋ┤уљєСИђСИІ№╝їжђЎТеБС╗ЦтЙїСИЇТюЃтєЇУИЕтѕ░уЏИтљїуџётЮЉсђѓ

тЁГуе« Scope жђЪУдй

Spring Bean уџётЁГуе« scope тѕєтѕЦТў»№╝џ

  1. singleton - жаљУеГ№╝їТЋ┤тђІ container тЈфТюЅСИђтђІт»дСЙІ
  2. prototype - Т»ЈТгА getBean жЃйТќ░т╗║СИђтђІт»дСЙІ
  3. request - Т»ЈтђІ HTTP request СИђтђІт»дСЙІ
  4. session - Т»ЈтђІ HTTP session СИђтђІт»дСЙІ
  5. application - ТЋ┤тђІ ServletContext СИђтђІт»дСЙІ
  6. websocket - Т»ЈтђІ WebSocket session СИђтђІт»дСЙІ

тЙїжЮбтЏЏуе«№╝ѕrequestсђЂsessionсђЂapplicationсђЂwebsocket№╝ЅтЈфтюе web application СИГтЈ»ућесђѓ

Singleton СйюућетЪЪ

жђЎТў»жаљУеГуџё scopeсђѓтдѓТъюСйаТ▓њТюЅуЅ╣тѕЦТїЄт«џ№╝їSpring т░▒ТюЃТіі Bean уЋХТѕљ singleton№╝џ

ТѕќУђЁТўјуб║ТїЄт«џ№╝џ

Singleton Bean тюе Spring container тЋЪтІЋТЎѓт░▒ТюЃУбФт╗║уФІ№╝ѕжаљУеГ№╝Ѕ№╝їСИдСИћТЋ┤тђІТЄЅућеућЪтЉйжђ▒ТюЪтЁДжЃйтЈфТюЅСИђтђІт»дСЙІсђѓТЅђС╗ЦтцџтђІтю░Тќ╣жЃйТ│етЁЦтљїСИђтђІ UserService№╝їТІ┐тѕ░уџёжЃйТў»тљїСИђтђІуЅЕС╗Хсђѓ

Prototype СйюућетЪЪ

Prototype т░▒СИЇСИђТеБС║є№╝їТ»ЈТгАСйатЙъ Spring ТІ┐ Bean№╝їжЃйТюЃухдСйаСИђтђІтЁеТќ░уџёт»дСЙІ№╝џ

жђЎтђІ Bean ТІ┐тЄ║СЙєт░▒Тў»ућет«їтЇ│СИЪ№╝їТЅђС╗ЦтЙѕжЂЕтљѕтГўТћЙСИђС║ЏУФІТ▒ѓуЏИжЌюуџёУЄеТЎѓУ│ЄТќЎсђѓ

Singleton Т│етЁЦ Prototype уџётЮЉ

жђЎУБАУдЂуЅ╣тѕЦТ│еТёЈсђѓтЂЄУеГСйаТюЅСИђтђІ singleton уџё service№╝їТЃ│УдЂТ│етЁЦСИђтђІ prototype уџё Bean№╝џ

уѓ║С╗ђж║╝ТюЃжђЎТеБ№╝ЪтЏауѓ║ Spring тюет╗║уФІ singleton Bean ТЎѓт░▒Тіі prototype уџё dependency Т│етЁЦжђ▓тј╗С║є№╝їтЈфТ│етЁЦСИђТгАсђѓС╣ІтЙїтЇ│Сй┐ prototype уџё scope УффУдЂТќ░т╗║№╝їСйєти▓уХЊУбФТ│етЁЦС║є№╝їт░▒Т▓њТюЅТќ░т╗║уџёТЕЪТюЃсђѓ

УДБТ│ЋСИђ№╝џObjectProvider

ObjectProvider ТЈљСЙЏС║є getObject() Тќ╣Т│Ћ№╝їТ»ЈТгАтЉ╝тЈФжЃйТюЃтљЉ Spring УдЂСИђтђІТќ░уџё Beanсђѓ

УДБТ│ЋС║ї№╝џ@Lookup

@Lookup жђЎтђІУе╗УДБтЉіУе┤ Spring№╝їжђЎтђІТќ╣Т│Ћуџёт»дуЈЙућ▒ Spring жђЈжЂј CGLIB С╗БуљєСЙєТЈљСЙЏсђѓТ»ЈТгАтЉ╝тЈФжЃйТюЃт╗║Тќ░т»дСЙІсђѓ

Request СйюућетЪЪ

Request scope уџё Bean тюеТ»ЈтђІ HTTP request уџёућЪтЉйжђ▒ТюЪтЁДтЈфТюЅСИђтђІт»дСЙІ№╝їrequest ухљТЮЪтЙїт░▒ТюЃУбФжіиТ»ђсђѓтИИућеСЙєтГўТћЙтќ«СИђ request уџёуЏИжЌюУ│ЄУеі№╝џ

ТѕќУђЁућеУе╗УДБТќ╣т╝Ј№╝џ

Session СйюућетЪЪ

Session scope уџё Bean тюетќ«тђІ HTTP session уџёућЪтЉйжђ▒ТюЪтЁДТюЅСИђтђІт»дСЙІ№╝їСИЇтљїуџё session ТюЃТюЅСИЇтљїуџёт»дСЙІсђѓтЙѕжЂЕтљѕтГўТћЙСй┐ућеУђЁуЏИжЌюуџёУ│ЄУеі№╝ѕжЏќуёХСИђУѕгжѓёТў»уће session attribute Т»ћУ╝Ѓтцџ№╝Ѕ№╝џ

Application СйюућетЪЪ

Application scope т░▒Тў»ТЋ┤тђІ ServletContext уџёућЪтЉйжђ▒ТюЪ№╝їжђџтИИућеТќ╝тГўТћЙтЁеТЄЅућетЁ▒С║ФуџёуІђТЁІ№╝џ

тЏауѓ║Тў»тЁеТЄЅућетЁ▒С║Ф№╝їтдѓТъюУдЂТћ╣У«іуІђТЁІуџёУЕ▒УдЂТ│еТёЈ thread-safeсђѓ

WebSocket СйюућетЪЪ

WebSocket scope тюеСй┐уће WebSocket ТЎѓ№╝їуѓ║Т»ЈтђІ WebSocket session т╗║уФІСИђтђІт»дСЙІсђѓжђЎтђІТ»ћУ╝Ѓт░Љуће№╝їжђџтИИтюет»дТЎѓТЄЅућеУБАТЅЇТюЃуб░тѕ░№╝џ

тдѓСйЋуће SpEL УфъТ│Ћ

ТюЅТЎѓтђЎСйаТЃ│УдЂуће SpEL№╝ѕSpring Expression Language№╝ЅСЙєУеГт«џ scope№╝їтЈ»С╗ЦжђЎТеБтЂџ№╝џ

тюе application.properties УБАУеГт«џ№╝џ

жЄЇж╗ъТЋ┤уљє

  • singleton№╝џжаљУеГ№╝їТЋ┤тђІ container СИђтђІт»дСЙІ№╝їТ│еТёЈ thread-safe
  • prototype№╝џТ»ЈТгАжЃйТќ░т╗║№╝їтИИућеТќ╝УЄеТЎѓуЅЕС╗Х
  • request/session/application/websocket№╝џweb application т░ѕуће
  • жЄЇж╗ъТЈљжєњ№╝џsingleton Т│етЁЦ prototype УдЂуће ObjectProvider Тѕќ @Lookup ТЅЇТюЃуюЪуџёТ»ЈТгАТќ░т╗║
  • тЙїСИЅуе« scope№╝ѕrequestсђЂsessionсђЂwebsocket№╝ЅТ»ћУ╝Ѓт░Љуће№╝їжђџтИИтЈфТюЅуЅ╣т«џта┤ТЎ»ТЅЇжюђУдЂ

СИІТгАт»ФуеІт╝ЈТЎѓ№╝їтЁѕТЃ│ТИЁТЦџжђЎтђІ Bean тѕ░т║ЋжюђУдЂС╗ђж║╝ scope№╝їСИђжќІтДІТЃ│тЦйт░▒СИЇТюЃтЪІтЮЉС║єсђѓ
