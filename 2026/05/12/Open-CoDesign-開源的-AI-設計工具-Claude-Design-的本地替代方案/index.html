йҖҷеҖӢе°ҲжЎҲ 2026 е№ҙ 4 жңҲжүҚе…¬й–ӢпјҢ18 еӨ©иЎқдёҠиҝ‘ 5,000 йЎҶжҳҹгҖӮдё»иҰҒй–ӢзҷјиҖ…жҳҜдёӯеңӢзҡ„ OpenCoworkAI еңҳйҡҠпјҢжҠҖиЎ“жЈ§жҳҜ Electron + React 19 + Vite 7 + TypeScript strict modeпјҢз”Ё Zustand з®ЎзӢҖж…ӢгҖҒBiome еҒҡ lintingгҖҒVitest и·‘жё¬и©ҰгҖҒPlaywright и·‘ E2EгҖӮ

еҰӮжһңдҪ д№ӢеүҚз”ЁйҒҺ Claude Design жҲ– Bolt.newпјҢй«”й©—дёҠйқһеёёжҺҘиҝ‘гҖӮдҪҶеёіжҳҜз®—еңЁдҪ иҮӘе·ұзҡ„ API key дёҠгҖӮ

жғіеғҸдҪ еңЁйӨҗе»іеҗғйЈҜгҖӮеӨҡж•ё SaaS иЁӯиЁҲе·Ҙе…·зҡ„жЁЎејҸжҳҜгҖҢеҗғеҲ°йЈҪжңҲиІ»еҲ¶гҖҚвҖ”вҖ”дёҚз®ЎдҪ йҖҷеҖӢжңҲеҒҡдәҶдёҖејөең–йӮ„жҳҜдёҖзҷҫејөпјҢеғ№ж јдёҖжЁЈгҖӮиҒҪиө·дҫҶеҲ’з®—пјҢдҪҶеҰӮжһңдҪ еҸӘжҳҜеҒ¶зҲҫйңҖиҰҒеҝ«йҖҹеҮәеҖӢ prototypeпјҢзӯүж–јжҜҸеҖӢжңҲиҠұйҢўйӨҠдёҖжҠҠеҫҲе°‘жү“й–Ӣзҡ„з‘һеЈ«еҲҖгҖӮ

Open CoDesign жҳҜгҖҢиҮӘеё¶йЈҹжқҗеҲ°е»ҡжҲҝгҖҚгҖӮBYOKпјҲBring Your Own Keyпјүж„Ҹе‘іи‘—дҪ жҺҘдёҠ ClaudeгҖҒGPTгҖҒGeminiгҖҒDeepSeekпјҢз”ҡиҮіжң¬ең°и·‘ OllamaпјҢз”ЁеӨҡе°‘з®—еӨҡе°‘гҖӮж”ҜжҸҙи¶…йҒҺ 20 зЁ®жЁЎеһӢпјҢжҸӣжЁЎеһӢе°ұж”№еҖӢдёӢжӢүйҒёе–®зҡ„дәӢгҖӮ

йҖҷдёҚеҸӘжҳҜзңҒйҢўзҡ„е•ҸйЎҢпјҢиҖҢжҳҜжҺ§еҲ¶ж¬ҠгҖӮдҪ зҡ„ promptгҖҒдҪ зҡ„иЁӯиЁҲз”ўеҮәгҖҒдҪ зҡ„е“ҒзүҢзҙ жқҗпјҢе…ЁйғЁз•ҷеңЁжң¬ж©ҹгҖӮе°Қж–јжҺҘжЎҲзҡ„иЁӯиЁҲеё«жҲ–иҷ•зҗҶе®ўжҲ¶ж•Ҹж„ҹиіҮж–ҷзҡ„еңҳйҡҠпјҢйҖҷеҖӢе·®з•°жҳҜж №жң¬жҖ§зҡ„гҖӮ

е…§е»әзҡ„иЁӯиЁҲжҠҖиғҪдёҚжҳҜеҷұй ӯ

Open CoDesign еә•еұӨи·‘зҡ„жҳҜеҹәж–ј pi-coding-agent зҡ„ agent иҝҙеңҲпјҢдҪҶи®“е®ғи·ҹдёҖиҲ¬ AI coding assistant жӢүй–Ӣи·қйӣўзҡ„жҳҜ 12 еҖӢ Design Skill ModulesгҖӮдҪ еҸҜд»ҘжҠҠйҖҷдәӣ skill зҗҶи§ЈжҲҗгҖҢй җиЁ“з·ҙеҘҪзҡ„иЁӯиЁҲе°Ҳе®¶гҖҚпјҡжңүе°Ҳй–ҖеҒҡ Slide Deck зҡ„гҖҒе°Ҳй–Җи“Ӣ Dashboard зҡ„гҖҒе°Ҳй–ҖеҲҮ Landing Page зҡ„гҖӮ

йҒёеҘҪ skill д№ӢеҫҢпјҢagent е°ұжңғеё¶и‘—е°ҚжҮүзҡ„ system prompt е’ҢиЁӯиЁҲзҙ„жқҹеҺ»з”ҹжҲҗгҖӮз”ўеҮәе“ҒиіӘжҜ”дҪ дёҹдёҖж®өиЈё prompt зөҰ ChatGPT еҘҪдёҖжҲӘпјҢеӣ зӮә skill иЈЎе·Із¶“з·ЁзўјдәҶжҺ’зүҲжҜ”дҫӢгҖҒиүІеҪ©е°ҚжҜ”гҖҒе…ғд»¶й–“и·қйҖҷдәӣиЁӯиЁҲеҹәжң¬еҠҹгҖӮ

15 еҖӢе…§е»ә Demo prompt д№ҹеҖјеҫ—зҺ©дёҖијӘпјҢеғҸжҳҜгҖҢSaaS pricing pageгҖҚжҲ–гҖҢanalytics dashboardгҖҚпјҢзӣҙжҺҘзңӢе®ғиғҪеҒҡеҲ°д»ҖйәјзЁӢеәҰгҖӮ

Comment жЁЎејҸе’Ң AI Sliders

йҖҷе…©еҖӢеҠҹиғҪжҳҜжҲ‘иҰәеҫ—еҒҡеҫ—жңҖиҒ°жҳҺзҡ„иЁӯиЁҲгҖӮ

Comment жЁЎејҸпјҡprototype з”ўеҮәеҫҢпјҢдҪ дёҚз”ЁеӣһеҺ»ж”№ promptгҖӮзӣҙжҺҘеңЁз•«йқўдёҠй»һжҹҗеҖӢжҢүйҲ•гҖҒжҹҗж®өж–Үеӯ—пјҢз•ҷдёӢдҝ®ж”№жҢҮзӨәвҖ”вҖ”гҖҢйҖҷеҖӢжҢүйҲ•ж”№жҲҗз¶ иүІгҖҚгҖҢйҖҷж®өжЁҷйЎҢеӯ—ж”ҫеӨ§гҖҚгҖӮAgent жңғйҮқе°ҚдҪ жЁҷиЁҳзҡ„е…ғзҙ еҺ»ж”№пјҢдёҚжҳҜж•ҙд»ҪйҮҚз”ҹгҖӮйҖҷеҖӢдә’еӢ•ж–№ејҸжҜ”гҖҢйҮҚж–°жҸҸиҝ°ж•ҙеҖӢз•«йқўгҖҚй«ҳж•ҲеӨӘеӨҡдәҶгҖӮ

AI Sliders жӣҙзӣҙиҰәгҖӮжӢүжЎҝиӘҝж•ҙйЎҸиүІйЈҪе’ҢеәҰгҖҒе…ғд»¶й–“и·қгҖҒеӯ—еһӢеӨ§е°ҸпјҢеҚіжҷӮеҸҚжҳ еңЁ prototype дёҠгҖӮдёҚз”ЁжҮӮ CSSпјҢдёҚз”ЁеҜ« padding: 24pxпјҢжӢүдёҖдёӢе°ұе®ҢдәӢгҖӮ

е°ҚпјҢе®ғд№ҹж”ҜжҸҙ DESIGN.mdвҖ”вҖ”дҪ еҸҜд»Ҙе®ҡзҫ©е“ҒзүҢиүІгҖҒеӯ—еһӢгҖҒй–“и·қиҰҸзҜ„пјҢд№ӢеҫҢжҜҸж¬Ўз”ҹжҲҗйғҪжңғйҒөе®ҲйҖҷеҘ—зі»зөұгҖӮйҖҷе°ҚйңҖиҰҒз¶ӯжҢҒиҰ–иҰәдёҖиҮҙжҖ§зҡ„е°ҲжЎҲдҫҶиӘӘпјҢзңҒдёӢзҡ„дҫҶеӣһдҝ®ж”№жҷӮй–“йқһеёёеҸҜи§ҖгҖӮ

еҢҜеҮәж јејҸеҫҲеҜҰйҡӣ

з”ўеҮәзҡ„ prototype еҸҜд»ҘеҢҜеҮәжҲҗ HTMLгҖҒPDFгҖҒPPTXгҖҒZIPгҖӮ

HTML зӣҙжҺҘдёҹзөҰе·ҘзЁӢеё«зңӢдә’еӢ•ж•ҲжһңгҖӮPDF зөҰиҖҒй—ҶзңӢйқңж…ӢзүҲйқўгҖӮPPTX жӢҝеҺ»жҸҗжЎҲгҖӮZIP жү“еҢ…ж•ҙд»ҪеҺҹе§ӢзўјгҖӮеӣӣзЁ®ж јејҸж¶өи“ӢдәҶеҫһгҖҢжҲ‘иҰҒ demo зөҰе®ўжҲ¶зңӢгҖҚеҲ°гҖҢжҲ‘иҰҒдәӨзөҰеүҚз«Ҝе·ҘзЁӢеё«еҒҡгҖҚзҡ„е®Ңж•ҙе ҙжҷҜгҖӮ

жңүдёҖиӘӘдёҖпјҢиғҪзӣҙжҺҘијёеҮә PPTX йҖҷй»һи »еҜҰз”Ёзҡ„гҖӮеҫҲеӨҡ prototype е·Ҙе…·еҲ°дәҶиҰҒжҸҗжЎҲзҡ„жҷӮеҖҷпјҢдҪ йӮ„жҳҜеҫ—жүӢеӢ•жҲӘең–иІј PowerPointпјҢOpen CoDesign зӣҙжҺҘе№«дҪ и·ійҒҺйҖҷжӯҘгҖӮ

macOS з”Ё Homebrewпјҡ

иЈқеҘҪд№ӢеҫҢеЎ«е…ҘдҪ зҡ„ API keyпјҢйҒёеҖӢжЁЎеһӢпјҢе°ұиғҪй–Ӣе§Ӣз”ЁдәҶгҖӮжІ’жңүиЁ»еҶҠжөҒзЁӢгҖҒжІ’жңүдҝЎз”ЁеҚЎз¶Ғе®ҡгҖҒжІ’жңү onboarding е•ҸеҚ·гҖӮ

дҪҶе®ғдёҚжҳҜиҗ¬иғҪзҡ„

е№ҫеҖӢдҪ и©ІзҹҘйҒ“зҡ„йҷҗеҲ¶гҖӮ

зӣ®еүҚжҳҜ v0.1.xпјҢе®үиЈқжӘ”йӮ„жІ’жңүзЁӢејҸзўјз°ҪеҗҚпјҢmacOS жңғи·іеҮәгҖҢз„Ўжі•й©—иӯүй–ӢзҷјиҖ…гҖҚзҡ„иӯҰе‘ҠпјҢдҪ еҫ—жүӢеӢ•еҲ°зі»зөұиЁӯе®ҡиЈЎе…ҒиЁұгҖӮйҖҷе°Қйқһе·ҘзЁӢеё«зҡ„дҪҝз”ЁиҖ…дҫҶиӘӘжҳҜеҖӢй–ҖжӘ»гҖӮ

е®ғдёҚжҳҜ Figma зҡ„жӣҝд»Је“ҒгҖӮжІ’жңүеҗ‘йҮҸз·ЁијҜгҖҒжІ’жңүең–еұӨжҰӮеҝөгҖҒжІ’жңүеӨҡдәәеҚіжҷӮеҚ”дҪңгҖӮе®ғеҒҡзҡ„дәӢжғ…жҜ”ијғеғҸжҳҜгҖҢеҝ«йҖҹжҠҠжғіжі•и®ҠжҲҗеҸҜдә’еӢ•зҡ„ HTML йӘЁжһ¶гҖҚпјҢиҖҢдёҚжҳҜгҖҢзІҫйӣ•зҙ°зҗўжҜҸеҖӢеғҸзҙ гҖҚгҖӮ

жІ’жңү undoгҖӮе°ҚпјҢ2026 е№ҙдәҶйӮ„жІ’жңү undoгҖӮж”№еЈһдәҶе°ұйҮҚж–°з”ҹжҲҗпјҢжҲ–иҖ…еҫһ comment жЁЎејҸеұҖйғЁдҝ®жӯЈгҖӮйҖҷдёҖй»һйңҖиҰҒзҝ’ж…ЈгҖӮ

жңҖеҫҢпјҢз”ўеҮәе“ҒиіӘй«ҳеәҰдҫқиіҙдҪ йҒёзҡ„жЁЎеһӢгҖӮClaude 3.5 Sonnet зҡ„еүҚз«Ҝз”ҹжҲҗиғҪеҠӣжҳҺйЎҜжҜ”е…¶д»–жЁЎеһӢеҘҪпјҢеҰӮжһңдҪ з”ЁжҜ”ијғе°Ҹзҡ„жЁЎеһӢпјҢзөҗжһңеҸҜиғҪжңғи®“дҪ еӨұжңӣгҖӮ

йҖҷд»¶дәӢиғҢеҫҢзҡ„е…©жўқз·ҡ

AI иЁӯиЁҲе·Ҙе…·йҖҷеҖӢй ҳеҹҹпјҢжӯЈеңЁжІҝи‘—е…©жўқз·ҡеҲҶиЈӮгҖӮ
