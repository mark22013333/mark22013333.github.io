жҜҸж”№дёҖеҖӢжӘ”жЎҲе°ұиҰҒжҢүдёҖж¬Ў YesгҖӮи·‘еҖӢ npm install иҰҒеҗҢж„ҸпјҢи®ҖеҖӢ .env иҰҒеҗҢж„ҸпјҢйҖЈ ls жңүжҷӮеҖҷйғҪиҰҒдҪ й»һдёҖдёӢгҖӮ

з”ЁйҒҺ Claude Code зҡ„дәәеӨ§жҰӮйғҪз¶“жӯ·йҒҺйҖҷзЁ®гҖҢеҗҢж„Ҹз–ІеӢһгҖҚгҖӮдҪ зҹҘйҒ“е®ғиҰҒеҒҡд»ҖйәјпјҢдҪ д№ҹдҝЎд»»е®ғпјҢдҪҶжөҒзЁӢе°ұжҳҜйҖјдҪ дёҖзӣҙжҢүзўәиӘҚгҖӮеҲ°еҫҢдҫҶдҪ ж №жң¬дёҚзңӢе…§е®№дәҶпјҢеҸҚе°„жҖ§ең°жҢү YesвҖ”вҖ”йҖҷжҷӮеҖҷжүҖи¬Ӯзҡ„е®үе…Ёж©ҹеҲ¶еҸҚиҖҢи®ҠжҲҗеҪўејҸгҖӮ

дёүжңҲ 24 иҷҹ Anthropic зҷјдҪҲзҡ„ Auto Mode е°ұжҳҜиЎқи‘—йҖҷеҖӢе•ҸйЎҢдҫҶзҡ„гҖӮ

Auto Mode еҒҡдәҶд»Җйәј

дёҖеҸҘи©ұпјҡи®“ Claude Code иҮӘе·ұеҲӨж–·дёҖеҖӢж“ҚдҪңе®үдёҚе®үе…ЁпјҢе®үе…Ёзҡ„е°ұзӣҙжҺҘеҹ·иЎҢпјҢдёҚе®үе…Ёзҡ„жүҚж“ӢдёӢдҫҶе•ҸдҪ гҖӮ

жҠҖиЎ“дёҠжҳҜдёҖеҖӢйӣҷеұӨеҲҶйЎһеҷЁпјҲtwo-layer classifierпјүгҖӮжҜҸеҖӢеӢ•дҪңеңЁеҹ·иЎҢеүҚжңғйҒҺе…©й—ңпјҡ

з¬¬дёҖй—ңжӘўжҹҘйҖҷеҖӢж“ҚдҪңжҳҜдёҚжҳҜдҪҝз”ЁиҖ…гҖҢеҗҲзҗҶй җжңҹгҖҚзҡ„иЎҢзӮәгҖӮдҪ еҸ«е®ғж”№ codeпјҢе®ғиҰҒеҜ«жӘ”жЎҲпјҢйҖҷжҳҜеҗҲзҗҶй җжңҹгҖӮдҪ еҸ«е®ғж”№ codeпјҢе®ғиҰҒи®ҖдҪ зҡ„ SSH keyпјҢйҖҷдёҚжҳҜгҖӮ

з¬¬дәҢй—ңжӘўжҹҘжңүжІ’жңү prompt injection зҡ„и·ЎиұЎгҖӮеҰӮжһңжңүдәәеңЁ codebase иЈЎеҹӢдәҶжғЎж„ҸжҢҮд»Өи©Ұең–и®“ Claude еҹ·иЎҢеҚұйҡӘж“ҚдҪңпјҢйҖҷеұӨиҰҒж“ӢдҪҸгҖӮ

йҖҡйҒҺе…©й—ңзҡ„ж“ҚдҪңиҮӘеӢ•ж”ҫиЎҢпјҢжІ’йҖҡйҒҺзҡ„йҷҚзҙҡеӣһжүӢеӢ•зўәиӘҚжЁЎејҸгҖӮ

еҜҰйҡӣж•ёж“ҡ

Anthropic е…¬дҪҲдәҶе…§йғЁжё¬и©Ұзҡ„зөҗжһңпјҡ

  • иӘӨж””зҺҮпјҲfalse positiveпјүпјҡ0.4% вҖ” ж„ҸжҖқжҳҜжҜҸ 250 еҖӢе®үе…Ёж“ҚдҪңиЈЎйқўпјҢеӨ§жҰӮжңү 1 еҖӢжңғиў«йҢҜиӘӨж“ӢдёӢдҫҶгҖӮ
  • жјҸж”ҫзҺҮпјҲfalse negativeпјүпјҡ5.7% вҖ” жҜҸ 18 еҖӢеҚұйҡӘж“ҚдҪңиЈЎйқўпјҢеӨ§жҰӮжңү 1 еҖӢжңғжәңйҒҺеҺ»гҖӮ

0.4% зҡ„иӘӨж””зҺҮе°ҚдҪҝз”Ёй«”й©—дҫҶиӘӘе№ҫд№Һз„Ўж„ҹгҖӮдҪҶ 5.7% зҡ„жјҸж”ҫзҺҮеҖјеҫ—еҒңдёӢдҫҶжғідёҖжғігҖӮ

еңЁ sandbox з’°еўғиЈЎпјҢжјҸж”ҫдёҖеҖӢеҚұйҡӘжҢҮд»Өй ӮеӨҡжҳҜз ҚжҺүдёҖдәӣдёҚи©Із Қзҡ„жӘ”жЎҲпјҢйҮҚе»әе°ұеҘҪгҖӮеңЁ production server дёҠпјҹйӮЈе°ұжҳҜеҸҰдёҖеҖӢж•…дәӢдәҶгҖӮйҖҷд№ҹжҳҜзӮәд»Җйәј Anthropic гҖҢе»әиӯ°гҖҚпјҲе…¶еҜҰе№ҫд№ҺжҳҜиҰҒжұӮпјүеҸӘеңЁ sandbox з’°еўғдҪҝз”Ё Auto ModeгҖӮ

жҖҺйәје•ҹз”Ё

зӣ®еүҚ Auto Mode жҳҜ research previewпјҢеҸӘй–Ӣж”ҫзөҰ Enterprise е’Ң API з”ЁжҲ¶пјҢйңҖиҰҒи·‘ Sonnet 4.6 жҲ– Opus 4.6гҖӮ

е•ҹз”Ёж–№ејҸеңЁ Claude Code иЁӯе®ҡиЈЎеҠ дёҖеҖӢ flagпјҡ

е•ҹз”ЁеҫҢпјҢClaude Code зҡ„иЎҢзӮәжЁЎејҸжңғеҫһгҖҢжҜҸжӯҘзўәиӘҚгҖҚи®ҠжҲҗгҖҢжңүйўЁйҡӘжүҚе•ҸгҖҚгҖӮдҪ жңғжіЁж„ҸеҲ°еӨ§йғЁеҲҶзҡ„жӘ”жЎҲи®ҖеҜ«е’Ң bash жҢҮд»ӨзӣҙжҺҘе°ұи·‘дәҶпјҢеҸӘжңүжҜ”ијғж•Ҹж„ҹзҡ„ж“ҚдҪңпјҲеҲӘйҷӨжӘ”жЎҲгҖҒеӯҳеҸ– credentialsгҖҒз¶Іи·Ҝи«ӢжұӮпјүжүҚжңғеҪҲеҮәзўәиӘҚгҖӮ

и·ҹ Hooks зҡ„жҗӯй…Қ

Auto Mode е’Ңд№ӢеүҚд»Ӣзҙ№йҒҺзҡ„ Hooks жҳҜдә’иЈңзҡ„гҖӮ

Hooks жҳҜзўәе®ҡжҖ§зҡ„иҮӘеӢ•еҢ–пјҡдҪ е®ҡзҫ©еҘҪиҰҸеүҮпјҢзү№е®ҡдәӢд»¶и§ёзҷјзү№е®ҡи…іжң¬гҖӮдёҚйқ  AI еҲӨж–·пјҢи©Іи·‘е°ұи·‘гҖӮ

Auto Mode жҳҜжҷәж…§зҡ„ж¬Ҡйҷҗз®ЎзҗҶпјҡAI иҮӘе·ұеҲӨж–·дёҖеҖӢж“ҚдҪңйңҖдёҚйңҖиҰҒдҪ еҗҢж„ҸгҖӮе®ғзҡ„жұәзӯ–её¶жңүдёҚзўәе®ҡжҖ§пјҢдҪҶжҸӣдҫҶзҡ„жҳҜжӣҙжөҒжҡўзҡ„е·ҘдҪңжөҒгҖӮ

е…©иҖ…з–ҠеңЁдёҖиө·зҡ„ж•ҲжһңпјҡHooks иҷ•зҗҶдҪ е·ІзҹҘиҰҒиҮӘеӢ•еҢ–зҡ„дәӢжғ…пјҲж”№е®ҢжӘ”жЎҲиҮӘеӢ• formatгҖҒcommit еүҚиҮӘеӢ•и·‘ lintпјүпјҢAuto Mode иҷ•зҗҶдҪ жҮ¶еҫ—дёҖеҖӢдёҖеҖӢжҢүеҗҢж„Ҹзҡ„ж—Ҙеёёж“ҚдҪңгҖӮ

жҗӯй…Қ Bypass жЁЎејҸзҡ„ PreToolUse Hook жӣҙе®үе…ЁвҖ”вҖ”еҚідҪҝ Auto Mode ж”ҫиЎҢдәҶдёҖеҖӢж“ҚдҪңпјҢдҪ зҡ„ Hook йӮ„жҳҜеҸҜд»Ҙж””жҲӘжҺүзү№е®ҡзҡ„еҚұйҡӘжҢҮд»ӨгҖӮзӯүж–јеӨҡдәҶдёҖеұӨдҝқйҡӘгҖӮ

е®ғеҲӨж–·зҡ„йӮҸијҜ

ж №ж“ҡ Anthropic зҡ„иӘӘжҳҺе’Ң TechCrunch зҡ„е ұе°ҺпјҢеҲҶйЎһеҷЁдё»иҰҒиҖғж…®йҖҷе№ҫеҖӢйқўеҗ‘пјҡ

иҮӘеӢ•ж”ҫиЎҢзҡ„е…ёеһӢж“ҚдҪңпјҡ

  • и®ҖеҸ–е°ҲжЎҲе…§зҡ„еҺҹе§ӢзўјжӘ”жЎҲ
  • еҜ«е…ҘдҪҝз”ЁиҖ…жҳҺзўәиҰҒжұӮдҝ®ж”№зҡ„жӘ”жЎҲ
  • еҹ·иЎҢ buildгҖҒtestгҖҒlint зӣёй—ңзҡ„жҢҮд»Ө
  • е®үиЈқе°ҲжЎҲе·Іе®Је‘Ҡзҡ„дҫқиіҙеҘ—д»¶

жңғж“ӢдёӢдҫҶзҡ„е…ёеһӢж“ҚдҪңпјҡ

  • еӯҳеҸ– credentialsгҖҒSSH keyгҖҒ.env зӯүж•Ҹж„ҹжӘ”жЎҲ
  • еҲӘйҷӨеӨ§йҮҸжӘ”жЎҲпјҲmass deletionпјү
  • зҷјиө·еӨ–йғЁз¶Іи·Ҝи«ӢжұӮ
  • дҝ®ж”№зі»зөұиЁӯе®ҡжҲ–е…Ёеҹҹй…ҚзҪ®
  • ж“ҚдҪңе…§е®№з–‘дјјдҫҶиҮӘеӨ–йғЁжіЁе…Ҙ

дёӯй–“ең°её¶зҡ„ж“ҚдҪңпјҲжҜ”еҰӮеҲӘйҷӨе–®дёҖжё¬и©ҰжӘ”жЎҲгҖҒеҜ«е…Ҙж–°зҡ„ config жӘ”жЎҲпјүеүҮиҰ–дёҠдёӢж–ҮиҖҢе®ҡгҖӮйҖҷиЈЎе°ұжҳҜйӮЈ 5.7% жјҸж”ҫзҺҮжңғеҮәзҸҫзҡ„ең°ж–№гҖӮ

е»әиӯ°зҡ„дҪҝз”Ёж–№ејҸ

йҒ©еҗҲй–Ӣе•ҹ Auto Mode зҡ„е ҙжҷҜпјҡ

  • жң¬ең°й–Ӣзҷјз’°еўғпјҢжҗӯй…ҚзүҲжң¬жҺ§еҲ¶
  • CI/CD pipeline иЈЎзҡ„иҮӘеӢ•еҢ–д»»еӢҷ
  • Sandbox з’°еўғеҒҡ POC жҲ– prototype
  • жңү Hooks еҒҡеә•еұӨйҳІиӯ·зҡ„е°ҲжЎҲ

дёҚе»әиӯ°й–Ӣе•ҹзҡ„е ҙжҷҜпјҡ

  • зӣҙжҺҘж“ҚдҪң production з’°еўғ
  • иҷ•зҗҶеҢ…еҗ«ж•Ҹж„ҹиіҮж–ҷзҡ„е°ҲжЎҲпјҲйҮ‘иһҚгҖҒйҶ«зҷӮпјү
  • е…ұз”Ёеёіиҷҹзҡ„ server
  • жІ’жңүзүҲжң¬жҺ§еҲ¶зҡ„зӣ®йҢ„

дёҖеҖӢеҜҰз”Ёзҡ„ patternпјҡеңЁ .claude/settings.json й–Ӣ Auto ModeпјҢеҗҢжҷӮеңЁ Hooks иЈЎиЁӯеҘҪ PreToolUse ж””жҲӘиҰҸеүҮгҖӮAuto Mode иІ иІ¬ж—ҘеёёжөҒжҡўеәҰпјҢHooks иІ иІ¬дҪ зө•е°ҚдёҚжғіиў«зў°еҲ°зҡ„еә•з·ҡгҖӮ

йҖҷд»ЈиЎЁд»Җйәј

Auto Mode дёҚеҸӘжҳҜдёҖеҖӢеҠҹиғҪжӣҙж–°пјҢе®ғеҸҚжҳ дәҶ AI agent ж¬Ҡйҷҗз®ЎзҗҶзҡ„жј”йҖІж–№еҗ‘гҖӮ

жңҖеҺҹе§Ӣзҡ„еҒҡжі•жҳҜгҖҢе…ЁйғЁжүӢеӢ•зўәиӘҚгҖҚвҖ”вҖ”е®үе…ЁдҪҶж…ўгҖӮжҺҘи‘—жҳҜгҖҢе…ЁйғЁиҮӘеӢ•гҖҚвҖ”вҖ”еҝ«дҪҶеҚұйҡӘгҖӮAuto Mode иө°зҡ„жҳҜз¬¬дёүжўқи·Ҝпјҡи®“ AI иҮӘе·ұеҒҡйўЁйҡӘи©•дј°пјҢеҸӘеңЁйңҖиҰҒзҡ„жҷӮеҖҷжү“ж–·дҪ гҖӮ

йҖҷеҖӢж–№еҗ‘дёҚеҸӘ Anthropic еңЁиө°гҖӮApple зҡ„ж–° Siri иҰҒеҒҡи·Ё App зҡ„иҮӘдё»ж“ҚдҪңгҖҒNVIDIA зҡ„ Agent Toolkit еңЁеҒҡеҸҜиҮӘжҲ‘жј”еҢ–зҡ„ runtimeгҖҒеҫ®и»ҹзҡ„ Agent 365 еңЁеҒҡзө„з№”зҙҡзҡ„ agent жІ»зҗҶгҖӮжүҖжңүдәәйғҪеңЁи§ЈеҗҢдёҖйҒ“йЎҢпјҡгҖҢжҖҺйәјзөҰ AI и¶іеӨ зҡ„иҮӘз”ұеәҰпјҢеҗҢжҷӮз¶ӯжҢҒдәәйЎһзҡ„жҺ§еҲ¶ж„ҹгҖӮгҖҚ

Auto Mode зҡ„зӯ”жЎҲжҳҜеҲҶеұӨзҡ„е®үе…Ёж©ҹеҲ¶гҖӮдёҚжҳҜйқһй»‘еҚізҷҪзҡ„й–Ӣй—ңпјҢиҖҢжҳҜдёҖеҖӢ spectrumвҖ”вҖ”еҫһе®Ңе…ЁжүӢеӢ•еҲ°е®Ңе…ЁиҮӘеӢ•д№Ӣй–“пјҢжңүеҫҲеӨҡдёӯй–“еҖјеҸҜд»ҘиӘҝгҖӮ
