AI 與科技新聞摘要 — 2026/03/27

發表於2026-03-27|更新於2026-03-27|AI與科技新聞
|總字數:1.2k|閱讀時間:4分鐘|瀏覽量:

本週 AI 圈過得不算平靜。Claude 拿到電腦使用權、OpenAI 買了一家資安新創、MCP 路線圖剛出爐、Magento 有個 CVSS 滿分漏洞正在被大規模利用。挑幾則比較有感的整理一下。

1. Claude 拿到你電腦的控制權——Dispatch + Computer Use 雙發

Anthropic 在 3 月 17 日推出 Dispatch，3 月 24 日再追加 Computer Use，兩個功能合在一起，Claude 現在可以接受你手機傳過來的任務，然後在你的桌機上自己開 App、點按鈕、填表單。

Dispatch 的概念很直白：手機是遙控器，Mac 是工作機。你用手機 app 丟任務，Claude 在桌機執行，跑完通知你。如果沒有直接整合的工具，它就走 Computer Use——截圖、點擊、輸入，靠視覺操控畫面完成任務。

幾個值得注意的細節：Claude 會先優先用現有連接器（Slack、Google Calendar 之類），只有工具不夠用才切換到視覺操控。有 prompt injection 偵測機制。目前 macOS 限定，Claude Pro / Max 方案。

Computer Use 官方也坦承「還是早期階段，比直接整合慢，也可能會犯錯」。這種誠實倒是挺少見的。

原文來源：Put Claude to work on your computer - Anthropic

2. GPT-5.4 上線：1M Context、Native Computer Use

OpenAI 在 3 月 5 日發佈 GPT-5.4，三個版本：標準版、Thinking（推理強化）、Pro（高效能）。

幾個數字：錯誤率比 GPT-5.2 降 33%、回應整體錯誤少 18%、部分任務 token 用量少 47%、API context window 最大到 100 萬 tokens。原生支援 Computer Use，可以截圖操控 GUI 執行多步驟任務。

同期 OpenAI 擴展了 ChatGPT 購物功能，引入 Agentic Commerce Protocol，加入 Walmart in-ChatGPT 體驗。但 Sora 平台在這段期間關閉了。

benchmark 跟實際用起來的落差向來不小，33% 誤差降低這個數字——參考一下就好。

原文來源：OpenAI launches GPT-5.4 with Pro and Thinking versions - TechCrunch

3. OpenAI 連買兩家：Promptfoo（AI 資安）+ Astral（Python 工具鏈）

Promptfoo 是做 AI 應用安全檢測的——幫企業在開發階段找 LLM 應用的漏洞和弱點。OpenAI 把它買下來，往 AI 安全方向走的意圖很明確。

Astral 是 3 月 19 日宣布的，做 Python 開發工具（uv、ruff 就是他們家的）。開發者社群對這個反應比較複雜：工具很好用，但被大公司買走之後會怎樣，大家心裡都有點問號。懂的都懂。

原文來源：OpenAI to acquire Promptfoo - OpenAI

4. MCP 2026 路線圖出爐：水平擴展、SSO、治理都在清單上

3 月 9 日 MCP 官方 Blog 發佈了 2026 路線圖。幾個主要方向：

Transport 擴展：Streamable HTTP 解決了生產環境部署問題，但水平擴展和無狀態操作還有缺口，下一代 transport 要補這塊。

企業級需求：audit trail、SSO 整合 auth、gateway 行為、設定可攜性——這些都是企業實際部署遇到的問題，路線圖明確列出要解決。

治理成熟化：Governance Working Group 要交付 Contributor Ladder SEP、Working Group 授權委派模型、公開 charter 模板。

On the Horizon：event-driven 觸發機制、streamed result type、更深層的安全與授權架構。

MCP SDK 的月下載量現在已經到了 9,700 萬次（2 月數據），生態系真的成形了。

原文來源：The 2026 MCP Roadmap - Model Context Protocol Blog

5. PolyShell 漏洞：Magento / Adobe Commerce 被大規模利用，CVSS 9.8

3 月 19 日開始，一個影響 Magento Open Source 和 Adobe Commerce 的漏洞 PolyShell 進入大規模利用階段。CVSS 評分 9.8，幾乎是滿分。未驗證的攻擊者可以上傳任意可執行檔。

跑電商的如果用這兩個平台，patch 沒打的話要趕快。CVSS 9.8 不是在嚇人——這不是靠修一個設定就能繞過的，是執行任意程式碼等級的問題。

原文來源：The Hacker News - Cybersecurity News

6. Xero 整合 Claude：記帳軟體遇上 AI

Xero 宣布深度整合 Anthropic Claude，把 AI 助手嵌入會計工作流。目標是讓小企業財務記帳流程自動化——分類費用、對帳、生成報表。

Claude 季增長 14.2%，ChatGPT 只有 4.1%。這個市場競爭速度快到有點嚇人。

原文來源：Xero and Anthropic Partner - World Today News

7. Google TurboQuant：KV Cache 壓縮 6x，速度快 8x，準確度沒掉

Google 發佈 TurboQuant，一個 LLM KV Cache 記憶體壓縮方案。數字很猛：至少 6 倍壓縮、最高 8 倍速度提升、準確度零損失。

KV Cache 一直是大 context window 的瓶頸，這個方向的進展對推論成本影響很大。還沒看到同行 reproduce 的結果，但如果數字是真的，這波頗躁。

本週摘要到這裡。MCP 生態系繼續在擴，Claude 和 GPT-5.4 都拿到電腦使用權，資安那塊該打 patch 的別拖。下週見。
