Spring Cloud Config Client иЁӯе®ҡиҲҮ @ConfigurationProperties

зҷјиЎЁж–ј2025-06-28|жӣҙж–°ж–ј2026-05-14|Spring
|зёҪеӯ—ж•ё:1.4k|й–ұи®ҖжҷӮй–“:6еҲҶйҗҳ|зҖҸиҰҪйҮҸ:

дёҠдёҖзҜҮж–Үз« и¬ӣдәҶ Config Server зҡ„иЁӯе®ҡпјҢйҖҷзҜҮе°ұдҫҶи¬ӣжҖҺйәјеңЁ client з«ҜжӯЈзўәең°жӢүе’ҢдҪҝз”ЁйҖҷдәӣиЁӯе®ҡгҖӮжңүдёҚе°‘дәәеңЁйҖҷйӮҠиё©йҒҺеқ‘пјҢдё»иҰҒйғҪжҳҜеӣ зӮәжҗһдёҚжё…жҘҡ bootstrap.yml е’Ң application.yml зҡ„еҚҖеҲҘгҖӮ

bootstrap.yml vs application.yml

йҖҷжҳҜжңҖй—ңйҚөзҡ„дёҖй»һпјҢеҚғиҗ¬еҲҘжҗһеҸҚдәҶгҖӮ

  • bootstrap.yml - з”ЁдҫҶиЁӯе®ҡ Spring Cloud зӣёй—ңзҡ„жқұиҘҝпјҢжҜ”еҰӮ Config Server зҡ„дҪҚзҪ®гҖӮе®ғзҡ„ијүе…Ҙе„Әе…ҲеәҸжҜ” application.yml жӣҙй«ҳ
  • application.yml - жҷ®йҖҡзҡ„жҮүз”ЁиЁӯе®ҡ

еҰӮжһңдҪ жҠҠ Config Server зҡ„иЁӯе®ҡеҜ«еңЁ application.ymlпјҢSpring жңғе…Ҳијүе…Ҙ application.ymlпјҢдҪҶйҖҷжҷӮеҖҷйӮ„жІ’жңүйҖЈеҲ° Config ServerпјҢжүҖд»ҘиЁӯе®ҡжңғдёҚе…ЁгҖӮдёҖе®ҡиҰҒеҜ«еңЁ bootstrap.ymlгҖӮ

Config Client зҡ„еҹәжң¬иЁӯе®ҡ

еңЁдҪ зҡ„ Spring Boot жҮүз”ЁиЈЎпјҢе…ҲеҠ дҫқиіҙпјҡ

з„¶еҫҢеңЁ bootstrap.yml иЈЎйқўиЁӯе®ҡпјҡ

еҗ„еҖӢиЁӯе®ҡй …зӣ®зҡ„еҗ«зҫ©

  • spring.application.name - жҮүз”Ёзҡ„еҗҚзЁұпјҢConfig Server жңғз”ЁйҖҷеҖӢеҗҚзЁұеҺ»жүҫе°ҚжҮүзҡ„иЁӯе®ҡжӘ”
  • spring.profiles.active - е•ҹз”Ёзҡ„ profileпјҢжҜ”еҰӮ devгҖҒprod
  • spring.cloud.config.uri - Config Server зҡ„дҪҚеқҖ

еҰӮжһңдҪ жғіиҰҒ Config Server зҡ„и·Ҝеҫ‘жӣҙиӨҮйӣңпјҢеҸҜд»ҘйҖҷжЁЈиЁӯе®ҡпјҡ

з”Ё @ConfigurationProperties з¶Ғе®ҡиЁӯе®ҡ

иҲҮе…¶з”Ё @Value йҖҗеҖӢжіЁе…ҘпјҢдёҚеҰӮз”Ё @ConfigurationProperties жҠҠзӣёй—ңзҡ„иЁӯе®ҡз¶ҒеңЁдёҖеҖӢ POJO class дёҠйқўгҖӮеҒҮиЁӯ Config Server жңүйҖҷжЁЈзҡ„иЁӯе®ҡпјҡ

дҪ еҸҜд»Ҙе»әз«ӢдёҖеҖӢиЁӯе®ҡ classпјҡ

з„¶еҫҢеңЁ service иЈЎйқўе°ұеҸҜд»ҘжіЁе…Ҙпјҡ

@ConfigurationProperties е’Ң @Value зҡ„еҚҖеҲҘ

зү№жҖ§ @ConfigurationProperties @Value

еүҚз¶ҙж”ҜжҸҙ ж”ҜжҸҙпјҢж•ҙеҖӢ prefix дёӢзҡ„йғҪжңғз¶Ғе®ҡ дёҚж”ҜжҸҙ
