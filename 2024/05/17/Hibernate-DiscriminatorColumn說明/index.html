Hibernate вҶ’ @DiscriminatorColumnиӘӘжҳҺ

зҷјиЎЁж–ј2024-05-17|жӣҙж–°ж–ј2024-09-22|Java
|зёҪеӯ—ж•ё:463|й–ұи®ҖжҷӮй–“:1еҲҶйҗҳ|зҖҸиҰҪйҮҸ:

@DiscriminatorColumn иЁ»и§Јз”Ёж–јеңЁ JPA з№јжүҝзөҗж§ӢдёӯпјҢзӮәе–®иЎЁзӯ–з•Ҙ (InheritanceType.SINGLE_TABLE) зҡ„з№јжүҝеұӨж¬Ўзөҗж§ӢжҢҮе®ҡеҚҖеҲҶеҲ—гҖӮйҖҷжЁЈпјҢJPA еҸҜд»ҘдҪҝз”ЁйҖҷеҖӢеҲ—дҫҶиӯҳеҲҘжҜҸдёҖиЎҢе°ҚжҮүзҡ„е…·й«”еҜҰй«”йЎһеһӢгҖӮ

и®“жҲ‘еҖ‘и©ізҙ°и§ЈйҮӢдёҖдёӢпјҡ

иЁ»и§ЈйғЁеҲҶ

  • @DiscriminatorColumnпјҡйҖҷеҖӢиЁ»и§ЈжҢҮе®ҡдәҶз”ЁдҫҶеҚҖеҲҶдёҚеҗҢеҜҰй«”йЎһеһӢзҡ„еҲ—гҖӮ
  • name="accountType"пјҡйҖҷжҢҮе®ҡдәҶеҚҖеҲҶеҲ—зҡ„еҗҚзЁұзӮә accountTypeгҖӮйҖҷеҖӢеҲ—е°Үжңғиў«ж·»еҠ еҲ°з”ҹжҲҗзҡ„е–®иЎЁдёӯпјҢз”ЁдҫҶеӯҳе„ІжҜҸдёҖиЎҢе°ҚжҮүзҡ„еҜҰй«”йЎһеһӢгҖӮ
  • discriminatorType = DiscriminatorType.STRINGпјҡйҖҷжҢҮе®ҡдәҶеҚҖеҲҶеҲ—зҡ„йЎһеһӢзӮә STRINGгҖӮеҚҖеҲҶеҲ—еҸҜд»ҘжҳҜ STRINGгҖҒCHAR жҲ– INTEGERгҖӮ

дҪңз”Ё

з•¶дҪҝз”Ё InheritanceType.SINGLE_TABLE з№јжүҝзӯ–з•ҘжҷӮпјҢжүҖжңүзҡ„з№јжүҝеұӨж¬Ўзөҗж§Ӣдёӯзҡ„еҜҰй«”йЎһеһӢйғҪжңғиў«жҳ е°„еҲ°еҗҢдёҖејөиЎЁдёӯгҖӮеҚҖеҲҶеҲ— (discriminator column) з”ЁдҫҶеҚҖеҲҶиЎЁдёӯзҡ„жҜҸдёҖиЎҢе°ҚжҮүе“ӘеҖӢе…·й«”зҡ„еҜҰй«”йЎһеһӢгҖӮдҫӢеҰӮпјҢеҰӮжһңжңүдёҖеҖӢ AdminGroup еҹәжң¬йЎһе’Ңе…©еҖӢеӯҗйЎһ AdminUser е’Ң AdminManagerпјҢйҖҷејөиЎЁдёӯзҡ„жҜҸдёҖиЎҢжңғжңүдёҖеҖӢ accountType еҲ—дҫҶжҢҮзӨәи©ІиЎҢжҳҜ AdminUser йӮ„жҳҜ AdminManagerгҖӮ

дҫӢеӯҗ

еҒҮиЁӯжҲ‘еҖ‘жңүд»ҘдёӢеҜҰй«”йЎһпјҡ

е°ҚжҮүзҡ„иіҮж–ҷеә«иЎЁеҸҜиғҪеҰӮдёӢпјҡ

id name accountType userSpecificField managerSpecificField
1 AdminUser1 USER someUserValue NULL
2 AdminManager1 MANAGER NULL someManagerValue

еңЁйҖҷеҖӢиЎЁдёӯпјҢaccountType еҲ—з”ЁдҫҶеҚҖеҲҶжҜҸдёҖиЎҢжҳҜ AdminUser йӮ„жҳҜ AdminManagerгҖӮ

зёҪзөҗ

@DiscriminatorColumn иЁ»и§Је‘ҠиЁҙ JPA еңЁе–®иЎЁзӯ–з•Ҙзҡ„з№јжүҝзөҗж§ӢдёӯпјҢдҪҝз”Ё accountType еҲ—дҫҶеҚҖеҲҶдёҚеҗҢзҡ„еҜҰй«”йЎһеһӢпјҢдёҰдё”йҖҷеҖӢеҲ—зҡ„йЎһеһӢжҳҜ STRINGгҖӮйҖҷжЁЈпјҢJPA е°ұеҸҜд»ҘжӯЈзўәең°е°ҮиЎЁдёӯзҡ„жҜҸдёҖиЎҢжҳ е°„еҲ°зӣёжҮүзҡ„еҜҰй«”йЎһгҖӮ
