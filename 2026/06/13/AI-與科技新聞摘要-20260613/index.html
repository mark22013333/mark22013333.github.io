AI 與科技新聞摘要 20260613 — Claude Code 再兩天變貴、Copilot 改用量計費、補貼派對散場

發表於2026-06-13|更新於2026-06-13|AI與科技新聞
|總字數:2k|閱讀時間:6分鐘|瀏覽量:

你每個月付的那筆 AI 訂閱費，到底是「商品的價格」，還是「一張被人替你墊著的帳單」？這個問題，這幾天突然從四個不同的方向，同時被擺到桌上。

一、Claude Code 再兩天就變貴，這次沒有反悔的餘地

6 月 15 日——也就是再兩天——Anthropic 早在 5 月 13 日就預告的計費改制正式生效。重點不在漲價，在「拆池子」。

在這之前，你的 Claude Code 和 Agent SDK 用量，跟你在 Claude.ai 聊天的額度共用同一個池子。6 月 15 日起，程式化的用量（claude -p、Agent SDK 那些自動跑的迴圈）被切出來，丟進一個獨立的月度信用額度池：Pro 20 美元、Max 5x 100 美元、Max 20x 200 美元，照完整 API 費率計算，用不完不能滾到下個月。

聽起來像會計細節，但對重度自動化使用者是實打實的痛。分析估算，重度 Agent SDK 使用者的有效成本會跳 5 到 10 倍；一份 20 美元的 Pro 額度，用 Sonnet 4.6 大概只夠跑 30 到 50 個中等規模的 coding agent 任務。

為什麼動這刀？有個數字解釋得很清楚：那些自動跑的 agent，過去等於從固定月費裡硬擠出 12 到 175 倍的補貼。換句話說，你掛在背景跑一整晚的那隻 agent，消耗的 token 帳面價值可能是你月費的上百倍——這缺口一直有人在墊。值得記的是，這已經是 Anthropic 今年第三次對程式化用量動手了：一月封掉訂閱 OAuth token 接第三方工具（被罵翻後幾天內收回），二月修改服務條款把 OAuth 認證限定在自家工具，現在是六月這刀。一次能反悔，兩次像試水溫，第三次連著同一個方向，那就不是失誤，是定調。

二、GitHub Copilot 早一步走同一條路，新用戶一度連門都進不去

巧的是，同一個方向，GitHub Copilot 6 月 1 日就先走了。

Copilot 整套改成用量計費，發給你「AI Credits」去扣。免費版 0 元、每月 2000 次補全加一小撮額度；Pro 10 美元換 1500 點；Pro+ 39 美元換 7000 點；再開一個 Max 方案 100 美元換 20000 點，附帶優先用新模型。混亂的是，改制上線那陣子，付費方案的新註冊一度被暫停，要等切換完才重開。

把這條跟第一條疊著看，方向一模一樣：「無限用」這三個字正在從這個產業集體消失。逆過來想最清楚——當兩家最大的玩家在兩週內、各自獨立地把「吃到飽」換成「按表計費」，那不是巧合，是同一個帳算到了同一個結論：吃到飽這頓飯，廚房快出不起了。

三、就在大家忙著漲價時，中國模型把價格砍到對手的三分之一

定價這場仗還有另一邊。社群拿 GPT-5、Claude Opus 4.8 跟 DeepSeek、Qwen 放在一起跑 benchmark，結論很刺眼：中國模型現在便宜了 3 到 10 倍。

更該注意的是這句話背後的轉向——2026 年 AI 的主旋律，正在從「誰更聰明」變成「誰更便宜」。過去兩年比的是 benchmark 多幾分，現在比的是同樣一件事誰能用更低的單位成本做完。當品質拉到「夠用就好」的水準後，差異化的戰場就從實驗室搬到了財務報表。這對前面兩條是雪上加霜：你才剛把吃到飽改成計費，想把真實成本轉嫁給用戶，隔壁就有人用三分之一的價格站在門口。能不能漲、漲多少，從來不是你一家說了算，是整桌人的成本結構說了算。

四、Anthropic 一邊遞件 IPO，一邊發報告談「AI 會不會搶走你的工作」

6 月 1 日，Anthropic 祕密遞件申請 IPO，OpenAI 也在排隊。同一個月，Anthropic 發了一份報告，標題叫《AI 對工作影響的政策框架》。

報告裡有個概念值得停一下：他們提出「observed exposure」——把模型理論上的能力，跟真實世界裡實際被用的程度兜在一起算，而不是只看「AI 理論上能取代什麼」。結論是目前實際覆蓋率還遠低於理論值。但真正犀利的是這句警告：失業率停在 5% 看起來一切正常，底下卻可能是勞動市場「換手」的速度，快過了人能重新適應的速度——AI 同時侵蝕各種職業的某些任務，表面數字卻完全看不出來。報告甚至提了個大膽的政策方向：給每個美國人一個「預分配資本帳戶」，讓人在 AI 重塑經濟的當口，手上先握有一份直接的股份。

一家公司一邊準備上市、要跟投資人講成長故事，一邊發報告提醒社會「這東西可能讓很多人沒工作」，這個並置本身就很值得玩味。但拋開動機，那個「平均數騙人」的提醒是對的：當一個系統的表面指標很漂亮、底層卻在劇烈換手，最危險的不是看得到的失業數字，是那些被平均值蓋住、等到浮上來已經來不及反應的結構性位移。

原文來源：A policy framework for AI’s impact on work, June 2026 — Anthropic、Microsoft / Anthropic AI 投資 — CNBC

五、Anthropic 最強的那個模型，只給 50 個夥伴用

最後一條剛好補上資安這塊，也補上一個反差。Anthropic 最新、最強的 Mythos 1，目前只開放給大約 50 個「Project Glasswing」夥伴，限定用途是防禦性的資安研究，一般開發者碰不到。

這個克制很罕見，也很合理。一個強到能拿來找漏洞、寫攻擊程式的模型，能力是雙面的——同一把能幫你補洞的鑰匙，反過來就能開別人的鎖。把它先關在 50 個受控的夥伴裡，等於承認了一件事：有些能力放出去之前，得先確定接住它的手是乾淨的。對照前四條那種「能接多少就接多少、能賣多便宜就賣多便宜」的擴張節奏，這條反而像個提醒——產業裡不是每一塊都在比誰跑得快，有些地方，慢，才是負責任。

原文來源：June 2026 AI Launch Wave: A Builder’s Decision Map — WaveSpeed

那個讓你用得很爽的價格，從來不是真的價格

把這五件事收束成一句話：過去兩年你習慣的那個 AI 價格，是一場補貼派對的入場費，不是商品的真實標價。

平台用創投的錢替你墊了 token 成本，換你養成依賴、養成「AI 隨便用」的習慣。習慣養成了，墊款也燒得差不多了，於是帳單開始一張張寄回真正的使用者手上——Claude Code 拆池子、Copilot 改計費、漲不動的就被更便宜的對手逼著面對成本。這不是哪家公司變壞了，是補貼這件事本來就有保存期限，到期是必然，只是沒人會提前替你標好日期。

所以與其等下一張帳單寄到才驚訝，不如現在就把問題換個方向問自己：如果今天沒有任何補貼，你手上這些 AI 工具，哪幾個你願意照「真實成本」付錢留下，哪幾個只是因為「反正現在很便宜」才一直開著？想清楚這題的人，等派對的燈全亮起來時，不會是那個還愣在原地、發現自己其實沒那麼需要它的人。
