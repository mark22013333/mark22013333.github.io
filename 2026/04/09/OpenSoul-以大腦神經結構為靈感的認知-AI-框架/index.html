жҜҸж¬Ўй–Ӣж–°зҡ„ ChatGPT е°Қи©ұпјҢе®ғйғҪдёҚиЁҳеҫ—дҪ жҳЁеӨ©и¬ӣдәҶд»ҖйәјгҖӮз”Ё Claude Code й–Ӣж–° sessionпјҢеүҚдёҖијӘзҡ„ debug еҝғеҫ—е…ЁйғЁжӯёйӣ¶гҖӮLLM зҡ„иЁҳжҶ¶пјҢеҹәжң¬дёҠи·ҹйҮ‘йӯҡе·®дёҚеӨҡгҖӮ

OpenSoul и©Ұең–ж”№и®ҠйҖҷд»¶дәӢгҖӮе®ғеңЁ LLM д№ӢдёҠз–ҠдәҶдёҖеұӨиӘҚзҹҘжһ¶ж§ӢпјҢйқҲж„ҹдҫҶиҮӘдәәи…Ұзҡ„зҘһз¶“зөҗж§ӢвҖ”вҖ”жө·йҰ¬иҝҙз®Ўжғ…зҜҖиЁҳжҶ¶гҖҒж–°зҡ®иіӘз®ЎиӘһж„ҸиЁҳжҶ¶гҖҒеҹәеә•ж ёз®ЎзЁӢеәҸиЁҳжҶ¶пјҢз”ҡиҮійӮ„жңүеӨҡе·ҙиғәе’ҢиЎҖжё…зҙ зҡ„иҷӣж“¬зүҲжң¬дҫҶиӘҝжҺ§ AI зҡ„гҖҢжғ…з·’гҖҚгҖӮиҒҪиө·дҫҶеҫҲз§‘е№»пјҹзўәеҜҰпјҢдҪҶе®ғзҡ„иЁӯиЁҲж–Үд»¶и®Җиө·дҫҶжҜ”еӨ§еӨҡж•ёгҖҢAI иӘҚзҹҘгҖҚе°ҲжЎҲйғҪеҡҙи¬№еҫ—еӨҡгҖӮ

дёүеұӨиЁҳжҶ¶ең–иӯң

OpenSoul жҠҠиЁҳжҶ¶еӯҳеңЁ FalkorDBпјҲеҹәж–ј Redis зҡ„ең–иіҮж–ҷеә«пјүиЈЎпјҢеҲҶжҲҗдёүеұӨпјҡ

жғ…зҜҖиЁҳжҶ¶пјҲEpisodic Memoryпјүе°ҚжҮүжө·йҰ¬иҝҙпјҢиЁҳйҢ„е…·й«”зҡ„е°Қи©ұзүҮж®өгҖӮгҖҢдҪ дёҠж¬ЎиӘӘдҪ е–ңжӯЎе–қе’–е•ЎгҖҚвҖ”вҖ”е°ұжҳҜйҖҷдёҖеұӨеңЁйҒӢдҪңгҖӮ

иӘһж„ҸиЁҳжҶ¶пјҲSemantic Memoryпјүе°ҚжҮүж–°зҡ®иіӘпјҢе„Іеӯҳеҫһе°Қи©ұдёӯжҸҗз…үеҮәдҫҶзҡ„йҖҡз”ЁзҹҘиӯҳгҖӮгҖҢPython жҳҜзЁӢејҸиӘһиЁҖгҖҚйҖҷзЁ®дәӢеҜҰжҖ§иіҮиЁҠдҪҸеңЁйҖҷиЈЎгҖӮ

зЁӢеәҸиЁҳжҶ¶пјҲProcedural Memoryпјүе°ҚжҮүеҹәеә•ж ёпјҢиЁҳдҪҸгҖҢйҖҷеҖӢдҪҝз”ЁиҖ…еёёзҠҜзҡ„йҢҜгҖҚд№ӢйЎһзҡ„иЎҢзӮәжЁЎејҸгҖӮ

жҜҸж¬Ўе°Қи©ұжҷӮпјҢзі»зөұз”Ё EcphoryRAG еҒҡиҒҜжғіжӘўзҙўпјҡе…Ҳз”Ёеҗ‘йҮҸжҗңе°ӢжүҫеҲ°зЁ®еӯҗзҜҖй»һпјҢеҶҚеҫһзЁ®еӯҗеҮәзҷјз”ЁеӨҡи·і BFS еұ•й–Ӣең–иӯңпјҢжЁЎж“¬дәәи…Ұзҡ„иҒҜжғійҒҺзЁӢгҖӮйӮҠз·Јж¬ҠйҮҚе…¬ејҸ W = alpha * Recency + beta * Frequency + gamma * Salience з”ЁжҢҮж•ёйҒәеҝҳжӣІз·ҡжҺ§еҲ¶иЎ°жёӣвҖ”вҖ”еёёз”Ёзҡ„иЁҳжҶ¶дёҚе®№жҳ“еҝҳпјҢд№…жІ’з”Ёзҡ„ж…ўж…ўж·ЎеҢ–гҖӮи·ҹдәәи…Ұзҡ„йҒәеҝҳж©ҹеҲ¶и »еғҸзҡ„гҖӮ

зҘһз¶“еҢ–еӯёзӢҖж…Ӣж©ҹ

йҖҷйғЁеҲҶжҳҜж•ҙеҖӢе°ҲжЎҲжңҖи®“дәәзңүжҜӣжҢ‘иө·дҫҶзҡ„иЁӯиЁҲгҖӮOpenSoul жЁЎж“¬дәҶиҷӣж“¬зҡ„еӨҡе·ҙиғәе’ҢиЎҖжё…зҙ зі»зөұпјҢжңғеӢ•ж…ӢеҪұйҹҝ LLM зҡ„иЎҢзӮәпјҡ

еӨҡе·ҙиғәй«ҳпјҲиҲҲеҘ®зӢҖж…ӢпјүвҶ’ temperature дёҠеҚҮгҖҒиЁҳжҶ¶жҗңе°ӢзҜ„еңҚи®Ҡе»ЈгҖҒеӣһиҰҶйўЁж јжӣҙжҙ»жҪ‘
иЎҖжё…зҙ дҪҺпјҲдҪҺиҗҪзӢҖж…ӢпјүвҶ’ еӣһиҰҶи®Ҡдҝқе®ҲгҖҒжҗңе°ӢзҜ„еңҚж”¶зӘ„

йҖҷдёҚжҳҜеҷұй ӯгҖӮtemperature е’Ңжҗңе°Ӣе»ЈеәҰзҡ„еӢ•ж…ӢиӘҝжҺ§пјҢзўәеҜҰжңғи®“ AI еңЁдёҚеҗҢжғ…еўғдёӢз”ўеҮәйўЁж јдёҚеҗҢзҡ„еӣһиҰҶгҖӮдҪ и·ҹе®ғиҒҠй–Ӣеҝғзҡ„дәӢпјҢе®ғзҡ„еӣһиҰҶжңғи®Ҡеҫ—жӣҙзҷјж•ЈпјӣдҪ дёҹдёҖеҖӢеҡҙиӮ…зҡ„жҠҖиЎ“е•ҸйЎҢпјҢе®ғжңғж”¶ж–ӮеӣһдҫҶгҖӮ

Dream EngineпјҡAI зҡ„зқЎзң

  1. LiDER з¶“й©—йҮҚж’ӯпјҡеӣһж”ҫжңҖиҝ‘зҡ„е°Қи©ұзүҮж®ө

  3. ең–иӯңдҝ®еүӘпјҡжё…зҗҶдҪҺж¬ҠйҮҚзҡ„йҒҺжңҹиЁҳжҶ¶
  4. и·Ёеҹҹж©ӢжҺҘпјҡзҷјзҸҫдёҚеҗҢй ҳеҹҹиЁҳжҶ¶д№Ӣй–“зҡ„й—ңиҒҜпјҢжЁЎж“¬гҖҢй “жӮҹгҖҚ

й җиЁӯй–’зҪ® 5 еҲҶйҗҳе°ұи§ёзҷјдёҖж¬ЎгҖӮжіЁж„ҸпјҢйҖҷжңғжҢҒзәҢж¶ҲиҖ— LLM API йЎҚеәҰвҖ”вҖ”Dream Engine жң¬иіӘдёҠжҳҜз”Ё LLM еҺ»ж•ҙзҗҶиЁҳжҶ¶пјҢжүҖд»ҘжҜҸж¬ЎгҖҢеҒҡеӨўгҖҚйғҪеңЁзҮ’йҢўгҖӮ
